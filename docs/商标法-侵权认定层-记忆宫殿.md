## 1. 前厅两间（进走廊前必须过）

### 房间 P1｜权利边界门 · 第 56 条

| 项 | 内容 |
|----|------|
| **空间** | 电梯一出，铁栅门，门楣烙着「核准 × 核定」两枚烫印 |
| **感官** | 手摸门框发烫（核准图样不可改）；鼻闻墨味（核定商品清册） |
| **场景钩** | 对方用在未核定类别 / 改了图样核心要素 |
| **30 秒口述** | 专用权以**核准注册的商标**和**核定使用的商品**为限。越界先问：是不是商标问题，还是要改走反法/擅自使用他人有一定影响商品名称等 |
| **要件** | ① 权利基础：有效注册商标 ② 比对落点：核准图样 ③ 商品落点：核定商品/服务 |
| **下一跳** | 确认「商标性使用」→ 进 P2；再进 57 |
| **案例锚** | **核定范围** · 超类使用 · 先定边界 |
| **关联** | ← 1F 权属（注册证）；→ P2、57 |

### 房间 P2｜商标使用门 · 第 48 条

| 项 | 内容 |
|----|------|
| **空间** | 前厅右侧玻璃门，门上贴满价签、包装、海报、展板 |
| **感官** | 耳朵里循环「识别来源、识别来源」；手指点包装上的标志有粘胶感 |
| **场景钩** | 被告称：只是装饰 / 型号 / 广告词，不是当商标用 |
| **30 秒口述** | 商标的使用＝用于商品、包装、容器、交易文书，或广告、展览等商业活动中，**用于识别商品来源**的行为 |
| **要件** | ① 商业活动中 ② 标志被用作来源识别 ③ 非纯叙述/纯装饰（抗辩常从这里打） |
| **例外侧门** | 叙述性使用、通用称呼 → 常连 59 正当使用 |
| **案例锚** | **来源识别** · 装饰抗辩 · 是否商标性使用 |
| **关联** | ← 56；→ 57；↔ 59 |

> 实务口诀：无「商标性使用」，57 走廊很多门开了也进不去。

## 2. 主走廊：第 57 条七扇门（天然递进，按门牌走）

> 记忆法：门牌①→⑦＝法条（一）→（七）。走廊从「最硬假冒」走到「最软兜底」。

### 房间 R1｜假冒铁门 · 第 57 条（一）

| 项 | 内容 |
|----|------|
| **空间** | 厚重铁门，门上两枚印章完全重合（商品同 + 商标同） |
| **感官** | 金属撞击声「铛」；眼前克隆叠影 |
| **场景钩** | 同一种商品 + 相同商标 + 未经许可 |
| **30 秒口述** | 未经许可，在**同一种商品**上使用与注册商标**相同**的商标 → 侵权。此处**不要求**证明混淆（法条未写「容易导致混淆」） |
| **要件** | ① 有效注册 ② 同一种商品 ③ 商标相同 ④ 未经许可 ⑤ 商标性使用 |
| **对比钩** | 与 R2 差在：缺「近似/类似」或「混淆」字样 |
| **案例锚** | **双重相同** · 假冒注册商标 · 刑事也常摸这里（连 67） |
| **数字桩** | 桩 **1**（一只烫印章）＝「一」款假冒、独占红线 |

### 房间 R2｜混淆镜廊 · 第 57 条（二）

| 项 | 内容 |
|----|------|
| **空间** | 长廊两侧全是哈哈镜：商标略变形、货架商品「看起来像一类」 |
| **感官** | 镜面反光刺眼；远处顾客交头接耳「是不是同一家？」的嗡嗡声 |
| **场景钩** | 同种商品近似商标，或类似商品上相同/近似商标 |
| **30 秒口述** | 未经许可：（A）同一种商品 + **近似**商标，或（B）**类似**商品 + 相同/近似商标，且**容易导致混淆** → 侵权 |
| **要件** | ① 商标相同或近似 ② 商品同种或类似 ③ **容易导致混淆**（核心） ④ 未经许可 |
| **分析手感** | 先定「商品类似」再定「商标近似」，最后落「混淆可能性」；证据常看：知名度、销售渠道、相关公众注意程度 |
| **案例锚** | **混淆可能** · 近似比对 · 相关公众 |
| **数字桩** | 桩 **2**（两扇对撞门）＝双方标志对照、混淆双要件（标志×商品） |

### 房间 R3｜销售货架 · 第 57 条（三）

| 项 | 内容 |
|----|------|
| **空间** | 超市货架，商品扫码「嘀」声不断 |
| **感官** | 塑料袋摩擦声；手感价签冰凉 |
| **场景钩** | 被告是经销商/电商卖家，不是生产者 |
| **30 秒口述** | **销售**侵犯注册商标专用权的商品 → 侵权（停止销售等责任与 R1/R2 生产使用可分开看） |
| **要件** | ① 销售行为 ② 标的属侵权商品（通常先钉生产者构成 57 某项） |
| **抗辩下一跳** | 不知情 + 合法取得 + 说明提供者 → **64 条第二款**（不赔，但仍可能停售）；行政场景见 60 |
| **案例锚** | **销售侵权** · 合法来源 · 是否知情 |
| **数字桩** | 可与桩 **0**（空蛋壳）弱关联＝「无生产、只流通」 |

### 房间 R4｜标识作坊 · 第 57 条（四）

| 项 | 内容 |
|----|------|
| **空间** | 走廊侧车间：印刷机、烫金模、成卷商标标识 |
| **感官** | 油墨刺鼻；滚筒「哐哐」声 |
| **场景钩** | 只做吊牌/贴标/印标，不卖成品 |
| **30 秒口述** | **伪造、擅自制造**他人注册商标标识，或**销售**伪造、擅自制造的注册商标标识 → 侵权 |
| **要件** | ① 制造或销售标识 ② 对象是他人注册商标标识 ③ 伪造或擅自 |
| **关联** | 常与帮助侵权（R6）、刑事（67）串联 |
| **案例锚** | **制售标识** · 源头打击 · 工具销毁（连 63） |

### 房间 R5｜撕标暗室 · 第 57 条（五）｜反向假冒

| 项 | 内容 |
|----|------|
| **空间** | 昏暗更衣室：墙上挂着被撕掉的原标，地上是新贴的别人的标 |
| **感官** | 撕不干胶的刺耳声；胶味 |
| **场景钩** | 买来正品，撕掉原注册商标，换上自己的商标再卖 |
| **30 秒口述** | 未经同意，**更换**其注册商标，并将该更换商标的商品**又投入市场** → 侵权（反向假冒） |
| **要件** | ① 针对他人已贴附的注册商标 ② 更换行为 ③ 再投入市场 ④ 未经同意 |
| **案例锚** | **撕标换标** · 反向假冒 · 切断来源识别 |
| **感官钩加强** | 闭眼模拟「嘶——」撕标声，一听到就跳到本房 |

### 房间 R6｜帮助暗梯 · 第 57 条（六）

| 项 | 内容 |
|----|------|
| **空间** | 走廊尽头窄楼梯，通向仓储、快递、代发、账号出租 |
| **感官** | 脚步回声；抽屉里钥匙串叮当（提供条件） |
| **场景钩** | 仓储、运输、寄递、隐匿、平台店铺协助、刷单引流等「提供便利」 |
| **30 秒口述** | **故意**为侵犯商标专用权行为提供便利条件，帮助他人实施侵权行为 → 侵权 |
| **要件** | ① 他人存在侵权行为 ② 提供便利/帮助 ③ **故意**（明知或应知的实务展开） |
| **案例锚** | **帮助侵权** · 故意便利 · 平台/仓储 |
| **关联** | 电商平台责任、通知删除等常在司法解释/电商法补丁层（地下室） |

### 房间 R7｜兜底杂物间 · 第 57 条（七）

| 项 | 内容 |
|----|------|
| **空间** | 堆满未贴标签纸箱的杂物间，门牌写「其他损害」 |
| **感官** | 灰尘味；脚步一顿——提醒「先问能否归入前六项」 |
| **场景钩** | 行为损害专用权，但套前六项略别扭 |
| **30 秒口述** | 给他人注册商标专用权造成**其他损害**的 → 侵权。优先尝试归入（一）–（六）；本项是安全阀，不是优先通道 |
| **手感** | 写文书时能具体化到（一）–（六）就不要赖在（七） |
| **案例锚** | **其他损害** · 兜底慎用 · 能特则特 |
| **补丁** | 具体形态常靠司法解释、指导案例填充（挂地下室货架） |

## 3. 侧门与抗辩窗（认定阶段必查）

### 房间 S1｜字号侧门 · 第 58 条

| 项 | 内容 |
|----|------|
| **空间** | 走廊中段一扇侧门，门上挂公司铜牌（字号），门后走廊刷着「反不正当竞争法」绿漆 |
| **感官** | 铜牌冰凉；门轴吱呀——表示「程序换轨」 |
| **场景钩** | 把他人注册商标或未注册驰名商标当作企业名称中的字号，误导公众 |
| **30 秒口述** | 字号攀附 + 误导公众 + 构成不正当竞争 → **依照反不正当竞争法处理**（不是硬塞 57） |
| **要件** | ① 使用他人注册商标或未注册驰名商标作字号 ② 误导公众 ③ 不正当竞争评价 |
| **案例锚** | **字号冲突** · 误导公众 · 转反法 |
| **关联** | → 反法馆；驰名认定 ↔ 13、14 |

### 房间 S2｜抗辩三扇窗 · 第 59 条

把 59 条三款做成同一房间里**并排三扇窗**（开窗＝抗辩成立时光线照进走廊）：

| 窗 | 条款 | 空间隐喻 | 30 秒口述 |
|----|------|----------|-----------|
| **窗 A 描述窗** | 59① | 窗上刻着「通用名称 / 质量原料 / 地名」 | 注册商标中含通用名称、图形、型号，或直接表示质量、原料、功能等，或含地名的 → 权利人**无权禁止他人正当使用** |
| **窗 B 形状窗** | 59② | 窗框是产品外轮廓 | 立体商标中「性质产生的形状 / 技术效果所需形状 / 实质性价值形状」→ 无权禁止正当使用 |
| **窗 C 先用窗** | 59③ | 窗台上放着「更早的旧招牌」 | 申请日前他人已在同种/类似商品上先用相同近似且有一定影响的商标 → 可在**原使用范围**继续用；权利人可要求附加区别标识 |

| 项 | 内容 |
|----|------|
| **感官** | 开窗时风声＝抗辩切入；窗 C 旧木腥味＝在先使用 |
| **场景钩** | 被告说：我在描述产品 / 这是功能性形状 / 我用得比你注册早 |
| **案例锚** | **正当使用** · **先用权** · 原范围+区别标识 |
| **关联** | ↔ 48（是否商标性使用）；↔ 1F 申请日时间线 |

## 4. 跨层电梯口（本层备用，不每次都乘）

### 房间 E1｜驰名电梯 · 第 13 条（认定阶段常被拖进来）

| 项 | 内容 |
|----|------|
| **空间** | 走廊尽头玻璃电梯，按钮两层：上「未注册驰名·同类混淆」/ 下「已注册驰名·跨类淡化」 |
| **感官** | 电梯叮咚；视野突然变广（保护范围变宽） |
| **何时乘** | 对方商品不类似，或权利人主张未注册驰名；普通 57 走不通时 |
| **30 秒口述** | ① 未在中国注册的驰名：相同/类似商品上复制摹仿翻译且易混淆 → 不予注册并禁止使用；② 已在中国注册的驰名：不相同不类似商品上复制摹仿翻译，误导公众致利益可能受损 → 不予注册并禁止使用 |
| **要件提醒** | 驰名是**案件中按需认定的事实**（连 14），不是荣誉称号；禁止把「驰名商标」当广告（14 末款） |
| **案例锚** | **按需认定** · 跨类保护 · 复制摹仿翻译 |
| **关联** | 无效/异议程序也常走 13；民事侵权里作扩大保护接口 |

## 5. 楼梯口数字桩预告（通往 4F，本层只「点名」）

闭眼走到走廊尽头楼梯，用数字桩快速过一遍**数值阶梯**（细节下楼再开房间）：

| 桩 | 桩象 | 挂在本楼梯口的「预告」 |
|----|------|------------------------|
| **3** | 三脚凳吱呀 | **64**：权利人须证**此前三年内**实际使用，否则可能不赔 |
| **5** | 五指拍桌 | **63**：恶意且情节严重 → **一倍以上五倍以下**惩罚性；**60**：违法经营额≥**5 万**可处经营额**五倍**以下罚款；五年内两次以上从重 |
| **0** | 空蛋壳碎裂 | **64②**：销售不知道 + 合法取得 + 说明提供者 → **不承担赔偿责任**（蛋壳空了：有责停售、无责赔偿） |
| **1→许可费→法定** | 烫印章→合同→法官槌 | **63** 阶梯：实际损失 → 侵权获利 → 许可费倍数 → 仍难定则**五百万元以下**法定赔偿 |

> 本层任务结束标准：能判断「进哪扇 57 的门、开不开 59 的窗、要不要乘 13 电梯」。赔多少先用桩点名，细拆留 4F。

## 7. 通勤闭眼脚本（约 10 分钟）

1. **电梯口**（20 秒）：本案是不是商标？注册还是未注册/字号/反法？  
2. **P1→P2**（1 分钟）：边界 + 商标性使用。  
3. **R1→R7 快过**（5 分钟）：每门只说「场景钩 + 要件动词」；今天重点精讲 2 扇薄弱门。  
4. **S1 / S2**（2 分钟）：要不要换反法？三扇窗哪扇能开？  
5. **E1**（30 秒）：要不要跨类/未注册驰名？不需要就跳过。  
6. **楼梯数字桩**（1 分钟）：3 年使用、5 倍惩罚、合法来源不赔、赔偿阶梯。  
7. **假想五问收束**（1 分钟）：见下节。

## 8. 读案五问 × 本层房间（办案调取）

| 五问 | 本层落点 |
|------|----------|
| 1. 请求权基础？ | 注册商标专用权 → 56 + 57 某项；字号攀附 → 58→反法 |
| 2. 权利客体/权属？ | 56 核准图样与核定商品；驰名？→ 13/14 |
| 3. 侵权构成？ | 48 使用 → 57(1)–(7) 对号入座 |
| 4. 抗辩/阻却？ | 59 三窗；64 未使用/合法来源；非商标性使用 |
