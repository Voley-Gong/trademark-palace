# 商标法 · 2F 授权与注册程序层｜完整记忆宫殿房间表

> 依据：《中华人民共和国商标法》（2019 年修正）  
> 定位：律所大楼「商标馆」二楼——申请、审查、异议、无效、撤三的「时间与程序走廊」  
> 下接 1F 权属客体 · 上通 3F 侵权认定  
> 数据：`docs/data/trademark-palace.json`

## 0. 本层总图

```text
电梯口（程序争议：还在申请？被异议？要无效？要撤三？）
    │
    ▼
申请段【填报】T1 22 ──►【优先权六个月】T2 25/26
    │
    ▼
审查段【九个月初审公告】T3 28 →【驳回】T4 30 →【驳回复审】T6 34
    │
    ▼
异议段【三月异议窗】T5 33 →【异议审理】T7 35 →【起算与追溯】T8 36
    │
    ▼
注册后纠错
  【绝对无效】T9 44 →【相对无效五年】T10 45 →【自始不存在】T11 47
  【撤三/通用名】T12 49 →（预告一年禁注 50）
```

**本层只解决一件事：** 走哪条程序、卡哪个期限、决定/裁定后权利状态如何。  
实体适格回 1F；侵权救济上 3F/4F。

## 1. 申请段

### 房间 T1｜分类填报台 · 第 22 条

| 项 | 内容 |
|----|------|
| **空间** | 柜台摆着商品分类表，一式申请单可勾多类 |
| **感官** | 纸张翻页声；电子提交「嘀」一声 |
| **场景钩** | 一类还是多类、商品名称怎么填 |
| **30 秒口述** | 按规定商品分类表填报类别和商品名称提出申请；一份申请可就多个类别申请同一商标；可书面或数据电文提出 |
| **案例锚** | **分类填报** · **一标多类** · 申请文件 |
| **关联** | → T2；核定范围终落 3F·56 |

### 房间 T2｜优先权沙漏 · 第 25、26 条

| 项 | 内容 |
|----|------|
| **空间** | 两个沙漏：外国首次申请 / 国际展览首次使用，都标「六个月」 |
| **感官** | 沙漏细沙声；逾期铃响「视为未要求」 |
| **场景钩** | 海外先申请再回国；展会上先用再申请 |
| **30 秒口述** | ① 国外首次申请日起**六个月内**就相同商品同一商标在中国申请，可享优先权（须声明并三个月内交副本）；② 中国政府主办或承认的国际展览会展出商品上首次使用，自展出日起**六个月内**可享优先权（须声明并三个月内交证明）。未声明或逾期 → 视为未要求优先权 |
| **数字桩** | **6 个月**优先权；**3 个月**补材料 |
| **案例锚** | **涉外优先权** · **展览优先权** · 逾期失权 |
| **关联** | ↔ 1F·O10 申请在先（优先权影响申请日地位） |

## 2. 审查段

### 房间 T3｜九个月公告廊 · 第 28 条

| 项 | 内容 |
|----|------|
| **空间** | 长廊尽头电子屏「初步审定公告」，倒计时九个月 |
| **感官** | 审查章咚一声；公告屏冷光 |
| **场景钩** | 申请后多久出结果、什么叫初审公告 |
| **30 秒口述** | 商标局应自收到申请文件之日起**九个月**内审查完毕；符合规定的，予以**初步审定公告** |
| **数字桩** | 桩联想：**9**＝九个月审查（可用自建桩） |
| **案例锚** | **九个月审查** · **初审公告** |
| **关联** | → T4 驳回；→ T5 异议窗打开 |

### 房间 T4｜驳回闸门 · 第 30 条

| 项 | 内容 |
|----|------|
| **空间** | 闸机落下：不符合规定，或与他人在同种/类似商品上已注册或初审的商标相同近似 |
| **感官** | 闸机金属撞击 |
| **场景钩** | 收到驳回通知、引证商标挡路 |
| **30 秒口述** | 不符合本法规定，或同他人在同一种/类似商品上已经注册或初步审定的商标相同近似的 → **驳回申请，不予公告** |
| **案例锚** | **绝对/相对驳回** · 引证商标 · 不予公告 |
| **关联** | → T6 复审；实体理由回 1F（10/11/32 等） |

### 房间 T6｜复审电梯 · 第 34 条

| 项 | 内容 |
|----|------|
| **空间** | 驳回闸门旁电梯：15 日上车 → 商评委九个月（可延三个月）→ 不服 30 日起诉 |
| **感官** | 电梯按键滴滴；期限钟滴答 |
| **场景钩** | 不服驳回怎么办 |
| **30 秒口述** | 不服驳回：收通知起**十五日内**向商标评审委员会申请复审；商评委原则上**九个月**决定（可延三个月）；不服复审决定：**三十日内**起诉 |
| **数字桩** | **15 日**复审；**9+3 月**审限；**30 日**诉讼 |
| **案例锚** | **驳回复审** · 审限 · 行政诉讼 |
| **关联** | ← T4；程序终点常接法院 |

> 房间号 T6 紧跟审查逻辑；T5 专留给异议窗（时间上初审公告后才开）。

## 3. 异议段

### 房间 T5｜三月异议窗 · 第 33 条

| 项 | 内容 |
|----|------|
| **空间** | 公告屏下三扇月历窗，窗楣「公告日起三个月」 |
| **感官** | 开窗风声；期满窗自动上锁「核准注册」 |
| **场景钩** | 发现近似标初审公告、要不要提异议 |
| **30 秒口述** | 初审公告之日起**三个月内**可异议：在先权利人/利害关系人可依 13②③、15、16①、30、31、32；**任何人**可依 4、10、11、12、19④。期满无异议 → 核准注册发证并公告 |
| **数字桩** | **3 个月**异议期（桩 3） |
| **案例锚** | **异议主体分流** · 三月期满 · 核准注册 |
| **关联** | → T7；绝对/相对事由回 1F |

### 房间 T7｜异议车间 · 第 35 条

| 项 | 内容 |
|----|------|
| **空间** | 双向传声筒：异议人 vs 被异议人；墙上流程箭头 |
| **感官** | 传声筒回音；决定书盖章 |
| **场景钩** | 异议后多久出决定、输了怎么办 |
| **30 秒口述** | 商标局听取双方、调查核实，自公告期满起原则上**十二个月**内决定（可延六个月）。准予注册 → 发证公告，异议人不服可依 44/45 请求无效。不予注册 → 被异议人**十五日内**复审（商评委原则上十二个月，可延六个月），不服**三十日内**起诉，法院通知异议人为第三人 |
| **案例锚** | **异议决定** · 准予后转无效 · 不予注册复审诉讼 |
| **关联** | → T8；→ T9/T10 |

### 房间 T8｜起算沙盘 · 第 36 条

| 项 | 内容 |
|----|------|
| **空间** | 沙盘两区：异议不成立获准注册的「权利起算点」；灰色地带「无追溯力」 |
| **感官** | 沙盘细沙；灰色区踩上去软（不能算旧账，恶意除外） |
| **场景钩** | 异议拖很久，专用权从哪天算；异议期间别人使用要不要赔 |
| **30 秒口述** | 期限届满不复审/不起诉 → 驳回或不予注册决定生效。异议不成立而准予注册的，专用权自**初审公告三个月期满之日**起算；自公告期满至准予注册决定前，对他人同种/类似商品上相同近似使用**原则上无追溯力**，但使用人**恶意**造成损失的应赔偿 |
| **案例锚** | **起算日** · **无追溯力** · 恶意例外 |
| **关联** | 侵权索赔时点常踩这里 |

## 4. 注册后：无效与撤销

### 房间 T9｜绝对无效炉 · 第 44 条

| 项 | 内容 |
|----|------|
| **空间** | 熔炉标签：第4、10、11、12、19④，或欺骗/不正当手段取得注册 |
| **感官** | 炉火灼热（绝对事由，可随时烧） |
| **场景钩** | 恶意注册、禁用禁注标已注册到手 |
| **30 秒口述** | 已注册商标违反 4/10/11/12/19④，或以欺骗及其他不正当手段取得注册 → 商标局可宣告无效；其他单位或个人可请求商评委宣告无效。后续有复审与诉讼通道 |
| **案例锚** | **绝对事由无效** · 商标局依职权 · 任何人可请求 |
| **关联** | ↔ 1F·O2/O5/O6；无五年限制（相对事由才有） |

### 房间 T10｜五年相对闸 · 第 45 条

| 项 | 内容 |
|----|------|
| **空间** | 闸门刻「注册之日起五年」；闸旁小门「驰名恶意不受五年限」 |
| **感官** | 闸门沉重；小门烫手（恶意） |
| **场景钩** | 在先权利被抢注、超过五年还能不能无效 |
| **30 秒口述** | 违反 13②③、15、16①、30、31、32 的已注册商标，在先权利人/利害关系人可自注册之日起**五年内**请求商评委无效；**恶意注册**的，**驰名商标所有人**不受五年限制 |
| **数字桩** | 桩 **5**＝五年除斥（与惩罚性五倍同桩不同场景） |
| **案例锚** | **相对无效** · **五年** · 驰名恶意例外 |
| **关联** | ↔ 1F·O8/O9；↔ 3F·E1 |

### 房间 T11｜自始空洞 · 第 47 条

| 项 | 内容 |
|----|------|
| **空间** | 注册证化作空框，匾额「视为自始即不存在」；角落保险箱「已执行的一般不追溯」 |
| **感官** | 空框回声；保险箱锁咔哒 |
| **场景钩** | 无效后过去判的侵权赔款、已付转让费怎么办 |
| **30 秒口述** | 宣告无效的注册商标，专用权**视为自始即不存在**。对宣告前已执行的侵权判决/行政处理、已履行的转让或许可，**原则上无追溯力**；但注册人恶意给他人造成损失应赔偿；不返还明显不公的，应全部或部分返还 |
| **案例锚** | **自始不存在** · **无追溯力** · 恶意与公平返还 |
| **关联** | ← T9/T10；影响 3F/4F 历史案件评估 |

### 房间 T12｜撤三磨坊 · 第 49 条

| 项 | 内容 |
|----|------|
| **空间** | 磨坊两石磨：①连续三年不使用；②注册标变成通用名称；旁侧小锤「擅自改变注册事项」 |
| **感官** | 石磨轰鸣；三年日历撕页 |
| **场景钩** | 对手注了不用；品牌沦为通用名；自行改变图样地址 |
| **30 秒口述** | 自行改变注册商标/名义地址等，责令改正，期满不改可撤销。注册商标成为核定商品**通用名称**，或**无正当理由连续三年不使用**，任何单位或个人可申请撤销；商标局原则上九个月决定（可延三个月） |
| **数字桩** | **3 年**不使用（与 64 三年使用证据同桩呼应） |
| **案例锚** | **撤三** · **通用名称撤销** · 改正后撤销 |
| **关联** | ↔ 4F·64 三年使用；↔ 1F·O6 显著/通用名；预告第50条一年内禁注相同近似 |

**楼梯口预告 · 第 50 条（点名即可）：**  
撤销、无效或期满不续展后，自该日起**一年内**，对相同或近似商标申请不予核准——防止「刚死立刻重生」搅局。

## 7. 通勤闭眼脚本（约 10–12 分钟）

1. **T1→T2**（1.5 分）：怎么申请、有没有优先权  
2. **T3→T4→T6**（3 分）：初审、驳回、复审期限链  
3. **T5→T7→T8**（3 分）：异议窗、审理、起算追溯  
4. **T9→T12**（3 分）：绝对/相对无效、自始、撤三  
5. **点名 50**（30 秒）：死后一年禁注  

口诀：**先看表（期限），再看门（事由），最后看坟（无效后还剩什么）。**
