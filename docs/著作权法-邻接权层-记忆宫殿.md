# 著作权法 · 邻接权侧廊｜完整记忆宫殿房间表

> 依据：《中华人民共和国著作权法》第 31–48、52、53 条  
> 定位：著作权馆 3F 侧廊——「与著作权有关的权利」（出版者 / 表演者 / 录音录像制作者 / 广播组织）  
> 入口：3F·R4 邻接权门厅  
> 对照：作品著作权走廊（P2/R3）；视听整片权属回 1F·O9  
> 数据：`docs/data/copyright-palace.json`

---

## 0. 本廊总图

```text
3F·R4 邻接权门厅
    │
    ├── N1 专有出版柜(33) · 版式设计窗(37)
    ├── N2 表演者权利束厅(39)
    ├── N3 职务表演岔路(40)
    ├── N4 表演保护期钟(41)
    ├── N5 录制者权利厅(44) ← 许可前置(42/43)
    ├── N6 录音制品二次获酬窗(45)
    ├── N7 广播组织禁止权厅(47)
    └── N8 广播播放许可规则(46/48)
```

**本廊只解决一件事：** 邻接权主体是谁、有哪些权、保护多久、使用时要过几道许可。  
作品是否侵权 → 回 R1/R3；整片影视归制作者 → O9；平台删帖 → 2F。

口诀：**邻接权不是作者权，出版表演录制广播四条柜；下游使用常要双重许可。**

## 1. 门厅

### 房间 R4｜邻接权门厅 · 第 53②–⑤、52⑨⑩（导览）

| 项 | 内容 |
|----|------|
| **空间** | 四扇柜门：专有出版 / 表演者 / 录制者 / 广播组织；门楣「与著作权有关的权利」 |
| **感官** | 唱片味、舞台追光、电波嘶嘶 |
| **30 秒口述** | 邻接权保护传播者投入，不等于作品著作权。侵权清单见 53②–⑤（专有出版、表演、录制、广播信号等）及 52 版式设计、现场表演等。进廊先认柜，再开具体房间 |
| **关联** | → N1–N8；↔ P2、R3 |

## 2. 出版者

### 房间 N1｜专有出版柜 · 版式窗 · 第 33、37 条

| 项 | 内容 |
|----|------|
| **空间** | 左柜「专有出版权」合同锁；右窗「版式设计」十年沙漏 |
| **感官** | 铅字油墨；沙漏十年 |
| **场景钩** | 一书两出版；扒版式出盗版书 |
| **30 秒口述** | 图书出版者对著作权人交付出版的作品，按合同约定享有的**专有出版权**受保护，他人不得出版该作品。出版者有权许可或禁止他人使用其图书、期刊的**版式设计**；保护期**十年**，截止于该图书、期刊首次出版后第十年 12 月 31 日 |
| **案例锚** | **专有出版** · **版式十年** · **合同约定** |
| **关联** | ↔ O8 委托；侵权→53②、52⑨ |

## 3. 表演者

### 房间 N2｜表演者权利束厅 · 第 39 条

| 项 | 内容 |
|----|------|
| **空间** | 舞台六盏灯：表明身份 / 形象不受歪曲 / 现场直播传送 / 录音录像 / 复制发行出租音像制品 / 信息网络传播 |
| **感官** | 追光烫脸；六盏灯可单独灭 |
| **场景钩** | 演唱会被偷录上网；歌手形象被恶搞剪辑 |
| **30 秒口述** | 表演者对其表演享有：①表明身份；②保护表演形象不受歪曲；③许可现场直播和公开传送现场表演并获酬；④许可录音录像并获酬；⑤许可复制、发行、出租录有其表演的录音录像制品并获酬；⑥许可信息网络传播其表演并获酬。被许可人以前述③–⑥方式使用时，**还应当取得著作权人许可并付酬** |
| **案例锚** | **六项表演者权** · **双重许可** · **形象不受歪曲** |
| **数字桩** | 桩 **2**＝表演者+著作权人双重许可 |
| **关联** | → N3、N4；侵权→53③、52⑩ |

### 房间 N3｜职务表演岔路 · 第 40 条

| 项 | 内容 |
|----|------|
| **空间** | 两条岔路：人身两项（表明身份、形象不受歪曲）留给演员；其他权利约定优先，未约定归**演出单位** |
| **感官** | 单位章与演员名牌对撞 |
| **场景钩** | 剧团演员离职后主张演出录像网播权 |
| **30 秒口述** | 完成本演出单位演出任务的表演为**职务表演**：演员享有表明身份和保护表演形象不受歪曲的权利；其他权利归属由当事人约定，无约定或约定不明的，由**演出单位**享有。若权利由演员享有，单位可在业务范围内免费使用该表演 |
| **案例锚** | **职务表演** · **人身两项保留** · **未约定归单位** |
| **关联** | ↔ O7 职务作品（另一套逻辑） |

### 房间 N4｜表演保护期钟 · 第 41 条

| 项 | 内容 |
|----|------|
| **空间** | 两面钟：人身两项「不受限制」；财产四项「表演发生后五十年」 |
| **感官** | 开演锣声起算五十年 |
| **30 秒口述** | 表明身份、保护表演形象不受歪曲的保护期**不受限制**。现场直播传送、录音录像、复制发行出租、信息网络传播等权利保护期为**五十年**，截止于该表演发生后第五十年 12 月 31 日 |
| **案例锚** | **人身无限期** · **财产五十年** · **表演发生起算** |
| **关联** | ↔ O12 |

## 4. 录音录像制作者

### 房间 N5｜录制者权利厅 · 第 42、43、44 条

| 项 | 内容 |
|----|------|
| **空间** | 制作台三步：向著作权人许可(42) → 与表演者订合同(43) → 录制者自有四权(44)：复制、发行、出租、信息网络传播 |
| **感官** | 母带冰凉；「首次制作完成」日期戳 |
| **场景钩** | 盗版碟、网盘传播整张专辑；出租录音制品 |
| **30 秒口述** | 制作录音录像制品应取得著作权人许可并付酬；录音制作者使用已合法录制为录音制品的音乐作品制作录音制品，可不经著作权人许可但应按规定付酬（声明不许除外）。制作时应与表演者订合同并付酬。录制者对制品享有许可他人**复制、发行、出租、信息网络传播**并获酬的权利，保护期五十年（首次制作完成后）。被许可人复制、发行、网传时还须同时取得著作权人、表演者许可；出租还须取得表演者许可 |
| **案例锚** | **录制者四权** · **制作前置许可** · **下游多重许可** |
| **关联** | ↔ N2；侵权→53④ |
| **侧签 42②** | 录音制品法定许可式制作（已合法录制音乐作品）——声明不许则不得用 |

### 房间 N6｜录音制品二次获酬窗 · 第 45 条

| 项 | 内容 |
|----|------|
| **空间** | 广播/公开播送窗口：把录音制品拿去有线无线公开传播或设备公开播送 → 向**录音制作者**付酬 |
| **感官** | 商场背景乐嗡嗡；付酬单据 |
| **场景钩** | 广播电台播歌、商场放碟 |
| **30 秒口述** | 将录音制品用于有线或无线公开传播，或通过传送声音的技术设备向公众公开播送的，应当向**录音制作者**支付报酬 |
| **案例锚** | **二次获酬** · **公开传播** · **录音制作者** |
| **关联** | ↔ T2 集体管理；与著作权人获酬规则可能并行 |

## 5. 广播组织

### 房间 N7｜广播组织禁止权厅 · 第 47 条

| 项 | 内容 |
|----|------|
| **空间** | 发射塔三道禁令：禁止擅自转播；禁止擅自录制复制；禁止擅自信息网络传播其播放的广播、电视 |
| **感官** | 电波干扰声；禁止牌红 |
| **场景钩** | 扒直播信号上网、转播卫视节目 |
| **30 秒口述** | 广播电台、电视台有权禁止未经许可：①以有线或无线方式**转播**其播放的广播、电视；②**录制以及复制**；③通过**信息网络向公众传播**。行使上述权利不得影响他人著作权或邻接权。保护期五十年，截止于该广播、电视**首次播放后**第五十年 12 月 31 日 |
| **案例锚** | **禁止转播** · **禁止录制复制** · **禁止网传信号** |
| **关联** | 侵权→53⑤；≠ 作品广播权（那是著作权人的） |

### 房间 N8｜广播播放许可规则 · 第 46、48 条

| 项 | 内容 |
|----|------|
| **空间** | 播控台两闸：未发表作品须许可+付酬；已发表作品可不经许可但应付酬。侧闸：电视台播视听作品/录像制品的许可链条 |
| **感官** | 闸机咔哒；付酬铃 |
| **场景钩** | 电台播小说、电视台播电影与录像 |
| **30 秒口述** | 广播电台、电视台播放他人**未发表**作品，应取得著作权人许可并付酬；播放**已发表**作品，可不经许可但应按规定付酬。电视台播放他人视听作品、录像制品，应取得视听作品著作权人或录像制作者许可并付酬；播放录像制品还应当取得著作权人许可并付酬 |
| **案例锚** | **未发表须许可** · **已发表可法定付酬** · **视听录像链条** |
| **关联** | ↔ O9、N5；与 N7 禁止权区分「台播别人」vs「别人动台的信号」 |

---

## 6. 总闸提醒 · 第 31 条

使用他人作品时，出版者、表演者、录制者、广播组织等**不得侵犯**作者的署名权、修改权、保护作品完整权和获得报酬权。邻接权再完整，也挡不住原作者人身权与获酬底线。

---

## 7. 数字桩

| 桩 | 挂载 |
|----|------|
| **2** | 表演者财产权使用常要过著作权人；录制下游多重许可 |
| **5** | 表演/录制/广播信号多档五十年；版式设计十年 |
| **0** | 职务表演未约定 → 财产性权利归演出单位 |

**30 秒假想案：**  
演唱会观众偷录完整上场并上传网盘收费 → 表演者 N2⑥ + 可能作品著作权人信息网络传播权；平台环节下 2F；赔偿上 4F。

*邻接权与作品著作权常并行主张；视听「整片」归制作者时，剧本音乐等可单独行权仍回 O9。*
