# 商标法 · 4F 救济层｜完整记忆宫殿房间表

> 依据：《中华人民共和国商标法》（2019 年修正）  
> 定位：律所大楼「商标馆」四楼——认定侵权之后，谈停、谈赔、谈程序  
> 上楼入口：3F 走廊尽头楼梯（见 `商标法-侵权认定层-记忆宫殿.md`）  
> 网页映射：与 3F 共用 `docs/data/trademark-palace.json`

## 0. 本层总图

```text
3F 楼梯上来
    │
    ▼
接待台【路径分流】60：协商 / 起诉 / 行政查处
    │
    ├── 左翼【行政车间】60②罚款阶梯、停售、没收销毁
    │         └─ 侧柜 61 查处移送 · 62 检查职权（略记）
    │
    ├── 中厅【赔偿殿】63
    │         四级阶梯走廊 → 惩罚性阁楼 → 举证抽屉 → 法定赔偿柜台 → 销毁后室
    │
    ├── 右翼【免赔/减责窗】64
    │         窗①三年未使用 · 窗②合法来源销售
    │
    └── 急诊通道【诉前】65 禁令+财产保全 → 66 证据保全
              └─ 红灯门【刑事】67（假冒/制标/明知销售）
```

**本层只解决一件事：** 走哪条救济路径、赔不赔、赔多少、来不来得及保全。  
侵权能不能成立 → 回 3F。

## 1. 接待台｜第 60 条（路径分流）

### 房间 F1｜分流接待台 · 第 60 条第一款 / 第三款

| 项 | 内容 |
|----|------|
| **空间** | 楼梯口接待台，三盏灯：**协商黄 / 法院红 / 行政蓝** |
| **感官** | 电话忙音；柜台木板被指尖敲得笃笃响（选路径） |
| **场景钩** | 刚认定 57 侵权，当事人问：报警？起诉？找市监？ |
| **30 秒口述** | 57 侵权纠纷：可协商；不愿或不成 → **起诉**或请求**工商（现市监）处理**。赔偿数额争议：可请行政部门**调解**，也可直接走民诉；调解不成或不履行 → 仍可起诉 |
| **要件/要点** | ① 民事与行政双轨并行可选 ② 行政侧重制止+罚款 ③ 赔偿谈不拢最终常回法院 |
| **案例锚** | **双轨救济** · 先协商 · 诉或查 |
| **关联** | → F2（行政罚则）；→ F3（民事赔偿）；← 3F·57 |

### 房间 F2｜行政车间 · 第 60 条第二款

| 项 | 内容 |
|----|------|
| **空间** | 左翼厂房：传送带运走侵权货，粉碎机轰鸣；墙上挂罚款价目表 |
| **感官** | 电机嗡嗡；粉尘味；扩音器喊「五年内两次，从重！」 |
| **场景钩** | 行政查处：停不停、罚多少、销售者能不能免罚式处理 |
| **30 秒口述** | 侵权成立 → 责令立即停止；没收、销毁侵权商品及主要用于制造侵权商品、伪造标识的工具。罚款：经营额**≥5 万**可处经营额**五倍以下**；无经营额或**<5 万**可处**25 万以下**。**五年内两次以上**或其他严重情节 → **从重**。销售不知道是侵权商品，能证明合法取得并说明提供者 → **责令停止销售**（行政场景下的「合法来源」） |
| **数字桩挂载** | 桩 **5**：五指拍桌＝**5 万**门槛 + **五倍**罚款 + **五年**再犯从重；桩 **2**：两扇门＝「两次以上」从重 |
| **案例锚** | **停售销毁** · 五倍罚款 · 五年再犯 |
| **关联** | ↔ F5 窗②（民事合法来源不赔）；→ 61 移送刑事 |

## 2. 中厅｜第 63 条（赔偿殿）——本层核心，按「阶梯走廊」走

> 把 63 条拆成同一大厅里的 **5 个连续站位**，用数字桩串顺序。闭眼只走这条走廊，赔偿规则就齐了。

```text
站①实际损失 → 站②侵权获利 → 站③许可费倍数
    → 站④惩罚性阁楼(1～5倍) → 站⑤法定赔偿柜台(≤500万)
         旁门：举证抽屉 · 销毁后室
```

### 房间 F3-①｜损失柜台 · 第 63 条第一款前段

| 项 | 内容 |
|----|------|
| **空间** | 大厅第一柜台，收银机打印「权利人少卖了多少」 |
| **感官** | 计算器按键声；账单纸涩手 |
| **口述** | 赔偿额**优先**按权利人因被侵权所受**实际损失**确定 |
| **桩** | 桩 **1**＝第一顺位 |

### 房间 F3-②｜获利柜台 · 第 63 条第一款中段

| 项 | 内容 |
|----|------|
| **空间** | 第二柜台，对面是侵权人的进销存屏 |
| **感官** | 屏幕蓝光刺眼 |
| **口述** | 实际损失难定 → 按侵权人因侵权所获**利益**确定 |
| **桩** | 桩 **2**＝第二顺位 |

### 房间 F3-③｜许可费柜台 · 第 63 条第一款后段

| 项 | 内容 |
|----|------|
| **空间** | 第三柜台，放着许可合同复印件 |
| **感官** | 合同碳粉味 |
| **口述** | 损失与获利都难定 → **参照商标许可使用费的倍数**合理确定。赔偿额应包括权利人为制止侵权支付的**合理开支** |
| **桩** | 桩 **3**＝第三顺位（许可费）；合理开支像贴在三个柜台上的「附加收据」 |

### 房间 F3-④｜惩罚性阁楼 · 第 63 条第一款惩罚段

| 项 | 内容 |
|----|------|
| **空间** | 柜台上方窄楼梯通阁楼，门牌「恶意 × 情节严重」 |
| **感官** | 上楼腿酸；阁楼热风扑面（恶意） |
| **口述** | 对**恶意**侵犯商标专用权且**情节严重**的，可在按上述方法确定数额的**一倍以上五倍以下**确定赔偿 |
| **要件** | ① 已能用损失/获利/许可费等方法定基数 ② 恶意 ③ 情节严重 ④ 倍数 1～5 |
| **数字桩** | 桩 **5**＝五倍上限；与行政 60 的「五倍」同桩，但场景不同：一民事惩罚、一行政罚款 |
| **案例锚** | **恶意严重** · 惩罚性 · 一至五倍 |

### 房间 F3-⑤｜法定赔偿柜台 · 第 63 条第三款

| 项 | 内容 |
|----|------|
| **空间** | 大厅尽端法官席柜台，铜牌「五百万元以下」 |
| **感官** | 法槌轻敲；铜牌冰凉 |
| **口述** | 实际损失、侵权获利、许可使用费**均难确定** → 法院依情节判决给予**五百万元以下**赔偿 |
| **数字桩** | 谐音/形象：桩 **5** 延伸为「500 万封顶」（拍桌五指 → 想起 500 万）；或自建桩「半扇门＝500」——选一个你记得住的，固定下来 |
| **案例锚** | **法定赔偿** · 情节酌定 · 五百万封顶 |

### 房间 F3-旁｜举证抽屉 · 第 63 条第二款

| 项 | 内容 |
|----|------|
| **空间** | 获利柜台旁的铁抽屉，钥匙在侵权人裤袋里 |
| **感官** | 抽屉锁嘎吱；空抽屉回声＝拒不提供 |
| **口述** | 权利人已尽力举证，而账簿资料主要由侵权人掌握 → 法院可责令侵权人提供；不提供或提供虚假 → 可**参考权利人主张和证据**定赔 |
| **案例锚** | **举证妨碍** · 账簿令 · 采信权利人主张 |
| **关联** | 实务常与申请书证令、证据保全（66）连用 |

### 房间 F3-后｜销毁后室 · 第 63 条第四、五款

| 项 | 内容 |
|----|------|
| **空间** | 大厅后门车间：假冒商品进粉碎口；模具丢进熔炉 |
| **感官** | 粉碎刺耳；焦糊味；门口告示「去标也不能回流」 |
| **口述** | 应权利人请求：假冒注册商标商品除特殊情况外**责令销毁**；主要用于制造的材料、工具责令销毁且**不予补偿**（特殊情况可禁入商业渠道且不补偿）。假冒商品**不得**仅去除假冒注册商标后进入商业渠道 |
| **案例锚** | **销毁不计补偿** · 禁止去标回流 |
| **关联** | ↔ 行政 60 没收销毁；↔ 57(1)(4) 假冒/制标 |

**63 条闭眼口诀（20 秒版）：**  
损→利→许可费 →（恶意严重则×1～5）→ 都没有就≤500 万；加合理开支；账簿可责令；假冒销毁、去标不准卖。

## 3. 右翼｜第 64 条（免赔两扇窗）

### 房间 F5｜免赔窗厅 · 第 64 条

两扇窗与 3F·59「抗辩三窗」同构，但功能是**赔不赔**，不是**侵不侵**。

| 窗 | 条款 | 空间 | 30 秒口述 | 数字桩 |
|----|------|------|-----------|--------|
| **窗① 休眠窗** | 64① | 窗台积灰的注册证，日历翻到「三年」 | 权利人请求赔偿，被告以「权利人未使用」抗辩 → 法院可要求权利人提供**此前三年内**实际使用证据；不能证明三年内使用，也不能证明其他损失 → 被告**不承担赔偿责任** | 桩 **3**＝三年 |
| **窗② 来源窗** | 64② | 窗边堆着进货单、发票、上家名片 | 销售不知道是侵权商品，能证明合法取得并说明提供者 → **不承担赔偿责任** | 桩 **0**＝空蛋壳（停得了、赔不出） |

| 项 | 内容 |
|----|------|
| **感官** | 窗①灰尘呛鼻＝三年未用；窗②发票油墨味＝合法来源 |
| **重要区分** | 不赔 ≠ 不构成侵权/不必停售。行政 60 对合法来源销售者是责令停售；民事 64②是免除赔偿责任 |
| **案例锚** | **三年使用** · **合法来源** · 免赔不免停 |
| **关联** | ← 3F·48（什么叫使用）；← 3F·R3 销售；↔ F2 行政合法来源 |

## 4. 急诊通道｜第 65、66 条（诉前）

### 房间 F6｜急症红门 · 第 65 条（行为禁令 + 财产保全）

| 项 | 内容 |
|----|------|
| **空间** | 大厅侧面急诊通道，红灯旋转；门后是「即将发货」的卡车 |
| **感官** | 警灯闪；引擎发动声；手心出汗（难以弥补） |
| **场景钩** | 侵权正在实施或即将实施，等判决来不及 |
| **30 秒口述** | 有证据证明他人正在实施或即将实施侵犯注册商标专用权的行为，不及时制止将使合法权益受到**难以弥补的损害** → 可依法在**起诉前**申请责令停止有关行为和**财产保全** |
| **要件** | ① 正在或即将实施 ② 商标侵权可能性 ③ 难以弥补的损害 ④ 起诉前 ⑤ 依法（担保等程序要件挂民诉法补丁） |
| **案例锚** | **诉前禁令** · 难以弥补 · 财产保全 |
| **关联** | → 起诉后行为保全；地下室：民诉法保全章节 |

### 房间 F7｜取样冰柜 · 第 66 条（诉前证据保全）

| 项 | 内容 |
|----|------|

| **感官** | 冷气扑面；封条撕开的脆响 |
| **场景钩** | 链接要下架、账簿要毁、样品要转移 |

| **要件** | ① 证据可能灭失/难取得 ② 与制止商标侵权相关 ③ 起诉前 |
| **案例锚** | **诉前证保** · 灭失风险 · 固定证据 |
| **关联** | ↔ F3 举证抽屉；实务常与公证、时间戳并行 |

## 8. 通勤闭眼脚本（约 8～10 分钟）

1. **接待台 F1**（30 秒）：本案走诉还是行政？赔要不要法院？  
2. **F2 罚款桩**（1 分钟）：5 万 / 五倍 / 25 万 / 五年两次。  
3. **F3 阶梯走廊**（4 分钟）：损→利→许可费→惩罚阁楼→500 万柜台；顺带开举证抽屉、销毁后室。  
4. **F5 两扇窗**（2 分钟）：三年使用？合法来源？强调「免赔不免停」。  
5. **F6→F7**（1.5 分钟）：来不来得及诉前？证据会不会没？  
6. **F8**（30 秒）：要不要亮红灯（明知/假冒/制标）？  
7. **串回 3F 五问第 5 问**（30 秒）：责任与程序答完即收功。

## 9. 读案五问 · 第 5 问展开（本层专用）

| 子问 | 落点 |
|------|------|
| 停不停？ | 60 行政责令 / 65 诉前禁令 / 判决停止侵害 |
| 罚不罚？ | 60② 经营额与倍数；五年再犯 |
| 赔不赔？ | 先 64 两窗；再进 63 |
| 赔多少？ | 63 五站 + 合理开支 + 举证妨碍 |
| 毁不毁？ | 60 没收销毁 / 63 销毁与禁回流 |
| 来得及吗？ | 65 / 66 |
| 要不要刑？ | 67 + 61 移送 |

**30 秒假想案：**  
经销商卖近似标鞋，进货有发票、声称不知假 → 3F 走 R2+R3；4F 开 F5 窗②争免赔，但仍可能停售；权利人三年无使用证据 → 再开窗①；要高额赔 → 走 F3 并找恶意证据冲阁楼；链接朝不保夕 → F7 冰柜。
