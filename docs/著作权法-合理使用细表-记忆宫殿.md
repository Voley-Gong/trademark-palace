# 著作权法 · 合理使用细表（第 24 条）

> 挂点：3F·S1 总厅 → B0·B17 细表柜  
> 口诀：**先对号入座，再过三句总闸；不付酬≠随便用。**  
> 数据：`docs/data/copyright-palace.json`

---

## 0. 三句总闸（任何一项情形都要过）

1. **指明**作者姓名或名称、作品名称（另有规定或无法指明的除外，办案仍默认先指明）  
2. **不得影响**该作品的正常使用  
3. **不得不合理地损害**著作权人的合法权益  

另：合理使用 = **不经许可、不付报酬**；适用于对邻接权的限制。  
对比 S2 法定许可（如教科书 §25）：通常仍要**付酬**。

## 1. 十三项对照表

| 项 | 关键词 | 能过倾向 | 难过倾向 |
|----|--------|----------|----------|
| **①** | 个人学习研究欣赏 | 私域、已发表、非面向公众传播 | 上传网盘公开分享、商业培训班整课使用 |
| **②** | 介绍评论说明·适当引用 | 引用为辅、有自己评论说明、署名 | 整段/整片搬运、引用变正文、无评论 |
| **③** | 报道新闻·不可避免再现 | 正规媒体报道所必需的片段 | 以新闻为名完整转播影视/演唱会 |
| **④** | 时事性文章转载播放 | 政经宗教时事性文章；他媒已发表 | 文艺娱乐长文；权利人声明不许 |
| **⑤** | 公众集会讲话 | 媒体刊登播放集会讲话 | 作者声明不许；非集会场景私采 |
| **⑥** | 课堂教学科研 | 少量、供教学科研人员；翻译改编汇编播放复制 | **出版发行**；面向不特定公众的营利网课整书复印 |
| **⑦** | 国家机关执行公务 | 合理范围、已发表、执行公务所需 | 机关工作人员私人自媒体搬运 |
| **⑧** | 馆藏陈列保存复制 | 本馆收藏、为陈列或保存版本 | 对外数字化公开提供替代市场 |
| **⑨** | 免费表演 | 未向公众收费、未向表演者付酬、不以营利为目的 | 门票/酒水捆绑、商业演出打「免费」旗号 |
| **⑩** | 公共场所艺术作品 | 临摹绘画摄影录像室外/公共陈列艺术作品 | 把摄影成果做成商品主图案大规模售卖（常另争） |
| **⑪** | 译少数民族语文 | 中国主体、国家通用语言文字作品→少数民族语文、**国内**出版发行 | 译成外语；主要在境外发行 |
| **⑫** | 阅读障碍者无障碍提供 | 无障碍方式、对象为阅读障碍者 | 借无障碍名义向公众大范围传播 |
| **⑬** | 法律行政法规其他情形 | 有明确另定依据 | 自行创造「合理」标签 |

## 2. 互联网高频卡点

| 场景 | 常主张 | 细表提醒 |
|------|--------|----------|
| 短视频封面/解说 | ②适当引用 | 看是否「适当」+有无独立评论；整集搬运难过 |
| 「个人欣赏」传网盘 | ① | 对不特定人分享≈传播，难靠① |
| 知识区讲书 | ②/⑥ | 商业账号讲全书 ≠ 课堂少量；⑥禁出版发行 |
| 新闻账号放完整赛事 | ③ | 「不可避免」是关键词，完整转播难过 |
| 商场背景音乐免费演 | ⑨ | 商场营利场所表演，难符合免费且非营利 |

## 3. 通勤与五问

**通勤：** R3/R1 认定使用 → S1 总闸 → B17 对号 →（付酬？）S2  

**五问：**  
① 落第几项？② 作品是否已发表（多项要求）？③ 三句总闸过不过？④ 有没有声明不许（④⑤）？⑤ 其实是法定许可要付酬？
