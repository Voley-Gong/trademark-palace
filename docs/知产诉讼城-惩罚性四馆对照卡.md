# 知产诉讼城 · 惩罚性四馆对照卡

> 网站入口：选馆页 / 各馆首页「惩罚性对照卡」  
> 数据：`docs/data/city-punitive.json`  
> 口诀：**故意（或恶意）+ 情节严重 → 基数 × 1～5 倍**

---

## 总闸（四馆共用）

1. 先定普通赔偿**基数**  
2. 再证**主观**（恶意 / 故意）  
3. 再证**情节严重**  
4. 基数上裁量 **1～5 倍**  
5. 合理开支另计；账簿举证妨碍常抬基数或情节  

**情节严重常见证据方向：** 重复侵权、规模化、隐匿伪造账簿、以侵权为业、持续时间长、影响/获利巨大。

## 五条对照记忆

1. **主观用词**：商标写恶意，其余多写故意。  
2. **范围最窄**：反法惩罚性目前主要钉**商业秘密**，混淆别乱主张惩罚性。  
3. **基数用词**：版权常说违法所得；其余多说侵权获利。  
4. **法定下限**：专利 3 万底、版权 500 元底，商标/反法多为 500 万以下酌定。  
5. **诉状写法**：主观事实一组 + 严重情节一组，分开举证。

④ 基数怎么定，账簿在谁手里？
