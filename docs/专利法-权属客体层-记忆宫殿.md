**本层只解决一件事：** 权利是否存在且有效、客体与文件是否立得住、权属与处分链条是否清楚。  
侵不侵权 → 上 3F；赔不赔 → 上 4F；稳不稳 → 上 2F。

## 1. 前厅：权利从哪来、是什么客体

### 房间 O1｜公告生效厅 · 第 39、40 条

| 项 | 内容 |
|----|------|
| **空间** | 大厅中央立着专利证书铜匣，匣盖刻「**公告之日 = 生效**」；匣侧两枚章：发明「实审后授」、实用/外观「初审后授」 |
| **感官** | 铜匣冰凉；启封「咔」一声（权利产生瞬间）；实审章更沉 |
| **场景钩** | 对方质疑：有证书但未公告？实用外观没实审算不算权？ |
| **30 秒口述** | 发明经实质审查无驳回理由 → 授予、发证、登记公告，专利权自**公告之日**起生效。实用新型、外观设计经初步审查无驳回理由 → 同样自**公告之日**起生效 |
| **要件** | ① 授权决定+登记公告 ② 生效时点＝公告日（不是申请日） ③ 发明有实审门槛，实用/外观初审授权（稳定性另走评价报告/无效） |
| **案例锚** | **公告生效** · **发明实审** · **实用外观初审** |
| **关联** | → O2；→ 3F·P1；稳定性 → 2F / 3F·E1 |
| **商标对照** | ≈ 商标 O1（核准→专用权）：都是「官方确认那一刻」生权 |

### 房间 O2｜三类客体仓 · 第 2 条

| 项 | 内容 |
|----|------|
| **空间** | 三排货架：发明（产品/方法/改进）· 实用新型（形状构造结合、适于实用）· 外观设计（整体或局部形状图案色彩、富有美感、工业应用） |
| **感官** | 方法货架飘着流程图味；外观货架灯光打在局部设计上 |
| **场景钩** | 本案到底是发明还是实用？外观保整体还是局部？方法专利怎么主张？ |
| **30 秒口述** | 发明创造＝发明、实用新型、外观设计。发明：产品、方法或其改进的新技术方案。实用新型：产品形状、构造或其结合的适于实用的新技术方案。外观设计：产品整体或局部的形状、图案或其结合以及色彩与形状、图案的结合，富有美感并适于工业应用的新设计 |
| **要件** | ① 先定类型 ② 实用新型须「产品」形状构造 ③ 外观可保局部 ④ 类型决定第11条行为清单与保护范围读法 |
| **案例锚** | **三类分流** · **方法在发明** · **局部外观** |
| **关联** | → O3/O4；→ 3F·P2（行为清单随类型变） |
| **商标对照** | ≈ 商标 O3 标志仓库：一边是「什么能当商标」，一边是「什么能当专利客体」 |

### 房间 O2b｜临时保护侧廊 · 第 13 条（挂标签精修）

| 项 | 内容 |
|----|------|
| **空间** | 前厅侧面玻璃廊：墙上日历停在「发明公布」；廊内不能贴禁令，只能收「适当费用」收据 |
| **感官** | 玻璃透亮但走不通（尚未完整独占实施权） |
| **30 秒口述** | 发明专利申请**公布后**，申请人可以要求实施其发明的单位或个人支付**适当的费用**（临时保护）；完整停止侵害等一般等授权公告（O1）后主张，时效起算见 4F·F4·74② |
| **案例锚** | **临时保护** · **适当费用** · **公布后** |
| **关联** | ↔ 4F·F4；公布程序细节 → 2F |

## 2. 客体走廊：能不能授、文件怎么钉死范围

### 房间 O3｜三性总纲门 · 第 22 条（发明 / 实用新型）

| 项 | 内容 |
|----|------|
| **空间** | 三扇连环门：新颖性 → 创造性 → 实用性；门楣挂「现有技术＝申请日前国内外公众所知」 |
| **感官** | 推门有阻力（创造性）；新颖性门缝吹来「抵触申请」冷风 |
| **场景钩** | 无效或抗辩攻击三性；抵触申请、现有技术抗辩与授权条件对照 |
| **30 秒口述** | 发明、实用新型须具备**新颖性、创造性、实用性**。新颖性：不属于现有技术，且无申请日前已申请、申请日后公布/公告的同样发明创造（抵触申请）。创造性：发明须突出的实质性特点+显著进步；实用新型须实质性特点+进步。实用性：能制造或使用并产生积极效果。现有技术＝申请日前国内外为公众所知的技术 |
| **要件** | ① 三性齐备 ② 现有技术时间点＝申请日 ③ 创造性标准发明严于实用 ④ 抵触申请杀新颖 |
| **案例锚** | **三性** · **现有技术** · **抵触申请** |
| **关联** | → O5；→ 2F 无效；↔ 3F·S1 现有技术抗辩（侵权侧「不构成侵权」） |
| **商标对照** | ≈ 商标 O4 显著总纲：都是「客体适格总门槛」 |

### 房间 O4｜外观门槛厅 · 第 23 条

| 项 | 内容 |
|----|------|
| **空间** | 展厅三道关卡：①不属于现有设计+无抵触 ②与现有设计或特征组合有明显区别 ③不与在先合法权利冲突 |
| **感官** | 灯光下对比现有设计册；墙角挂著作权/商标铭牌（在先权） |
| **场景钩** | 外观无效；撞了别人在先著作权或商标图样 |
| **30 秒口述** | 外观设计须不属于现有设计，且无申请日前同样申请、申请日后公告的抵触；与现有设计或现有设计特征组合相比应有**明显区别**；不得与他人申请日前已取得的**合法权利**相冲突。现有设计＝申请日前国内外公众所知的设计 |
| **案例锚** | **非现有设计** · **明显区别** · **在先权利冲突** |
| **关联** | → O7；→ 3F·R4；在先权冲突常连版权/商标 |
| **商标对照** | 外观「在先权利冲突」≈ 商标 O4/O9 的在先权思路 |

### 房间 O5｜不授予红柜 · 第 5、25 条

| 项 | 内容 |
|----|------|
| **空间** | 上锁红柜两层：上层「违法/公德/公益/遗传资源」；下层六格：科学发现、智力规则、疾病诊治、动植物品种、原子核变换、平面印刷品标识性设计 |
| **感官** | 红漆刺鼻；柜锁冰冷；动植物格旁小字「生产方法可授」 |
| **场景钩** | 商业方法、诊断方法、品种、主要起标识作用的平面设计 |
| **30 秒口述** | 违反法律、社会公德或妨害公共利益的，不授予；违法获取/利用遗传资源并依赖其完成的，不授予。另有六类不授予：科学发现；智力活动规则方法；疾病诊断和治疗方法；动植物品种（但其**生产方法**可授）；原子核变换方法及所得物质；对平面印刷品图案色彩等主要起**标识作用**的设计 |
| **案例锚** | **绝对排除** · **诊治方法** · **标识性平面设计** |
| **关联** | 无效 absolute 事由 → 2F；遗传资源说明 ↔ O6 文件柜 |
| **商标对照** | ≈ 商标 O5 禁用 + O6 禁注的「绝对事由柜」合体 |

### 房间 O5b｜新颖性宽限期侧窗 · 第 24 条（挂标签）

| 项 | 内容 |
|----|------|
| **空间** | 红柜旁小窗，窗框刻「申请日前**六个月**」 |
| **口述** | 紧急状态/非常情况为公益首次公开；中国政府主办或承认的国际展览会首次展出；规定学术会议首次发表；他人未经同意泄露 → 不丧失新颖性 |
| **数字桩** | 桩 **6**＝六个月宽限（自建：六面骰） |
| **关联** | ↔ O3 新颖性；细节口径 → B0/2F |

### 房间 O6｜权要·说明书文件柜 · 第 26 条（本层核心文件房）

| 项 | 内容 |
|----|------|
| **空间** | 铁柜四屉：请求书 / 说明书（清楚完整、本领域技术人员能够实现）/ 摘要 / **权利要求书**（以说明书为依据，清楚简要限定保护范围） |
| **感官** | 权要抽屉最重；说明书纸张沙沙（实现可能性）；遗传资源来源栏要手写 |
| **场景钩** | 保护范围争解释、说明书是否支持、能否实现、修改超范围（33 预告 2F） |
| **30 秒口述** | 发明实用新型申请须提交请求书、说明书及摘要、权利要求书。说明书须清楚、完整，以所属技术领域技术人员**能够实现**为准。权利要求书须以说明书为**依据**，清楚、简要限定保护范围。依赖遗传资源的，说明直接来源与原始来源（不能说明原始来源的陈述理由） |
| **要件** | ① 文件齐全 ② 充分公开/能够实现 ③ 权要以说明书为依据 ④ 清楚简要 |
| **案例锚** | **充分公开** · **权要依据说明书** · **限定保护范围** |
| **关联** | → 3F·P1/R1/R2（64 以权要为准、说明书解释）；捐献/禁止反悔 → 3F·W1 |
| **商标对照** | 商标无直接对应；这是专利「客体如何被写进权利」的心脏房 |
| **数字桩** | 桩 **1**＝权要红线（与 3F 字面覆盖同桩） |

### 房间 O7｜外观视图厅 · 第 27 条

| 项 | 内容 |
|----|------|
| **空间** | 环形展柜：图片/照片必须清楚显示要求保护的产品外观；旁挂「简要说明」小牌 |
| **感官** | 灯光均匀；转盘看清局部设计 |
| **场景钩** | 视图不清、简要说明与视图关系、局部外观如何显示 |
| **30 秒口述** | 外观设计申请须提交请求书、图片或照片、简要说明等；图片或照片应**清楚地显示**要求专利保护的产品的外观设计 |
| **案例锚** | **视图清楚** · **简要说明** · **显示保护对象** |
| **关联** | → 3F·R4；保护范围读法 → 64② |
| **商标对照** | 类似「注册图样要清晰」，但外观以视图本身定范围 |

## 3. 权属侧廊：是谁的

### 房间 O8｜职务发明厅 · 第 6 条（+15 奖励标签）

| 项 | 内容 |
|----|------|
| **空间** | 单位办公室：任务令 + 主要物质技术条件清单；墙上合同夹「约定优先」；抽屉放奖励/报酬通知书（15） |
| **感官** | 工牌冰凉；合同纸张涩手 |
| **场景钩** | 离职员工争权属；主要利用单位设备；合同约定归属 |
| **30 秒口述** | 执行本单位任务，或**主要**利用本单位物质技术条件完成的 → 职务发明；申请权属单位，批准后单位为专利权人，单位可依法处置。非职务 → 发明人/设计人。利用单位条件但有合同约定归属的，**从其约定** |
| **侧签 15** | 单位应对职务发明人/设计人给予**奖励**；实施后给予合理**报酬**；鼓励股权期权分红等产权激励 |
| **案例锚** | **职务发明** · **约定优先** · **奖励报酬** |
| **关联** | ↔ O9；发明人署名权 → 16 |
| **商标对照** | 商标少「职务创作」专条密度；专利权属争这里极常见 |

### 房间 O9｜合作委托厅 · 第 8 条

| 项 | 内容 |
|----|------|
| **空间** | 双人签约桌：合作完成 / 委托完成两套合同模板；缺协议时桌灯亮「完成方」 |
| **感官** | 笔尖停顿（有没有协议） |
| **场景钩** | 共同开发谁去申请；委托研发默认权属 |
| **30 秒口述** | 合作完成或接受委托完成的发明创造，**除另有协议外**，申请专利的权利属于完成或共同完成的单位/个人；批准后，申请的单位/个人为专利权人 |
| **案例锚** | **协议优先** · **完成方申请** · **合作共完** |
| **关联** | → O10 共有行使；→ O12 转让 |
| **商标对照** | 类似商标共有/承揽约定，但专利本条更日常 |

### 房间 O10｜共有议事厅 · 第 14 条

| 项 | 内容 |
|----|------|
| **空间** | 圆桌议事：中央牌「有约定从约定」；两扇绿灯「单独实施 / 普通许可」；红灯门「其余行使须全体同意」 |
| **感官** | 绿灯易推；红灯门沉重（转让、独占许可等） |
| **场景钩** | 一共有人自己实施；一共有人普通许可收使用费；要转让或独占许可卡住 |
| **30 秒口述** | 共有人有约定从约定。无约定：共有人可**单独实施**或以**普通许可**许可他人实施；普通许可使用费应在共有人之间分配。**除此以外**行使共有专利申请权或专利权，须取得**全体共有人同意** |
| **要件** | ① 约定优先 ② 默认可单独实施/普通许可 ③ 其他处分要全体同意 ④ 许可费内部分配 |
| **案例锚** | **单独实施** · **普通许可** · **其余须共有同意** |
| **关联** | ← O9；维权原告适格常连解释（B0） |
| **商标对照** | 商标共有规则不同；专利此厅务必单记 |

### 房间 O10b｜先申请 / 禁止重复授权侧签 · 第 9 条

| 项 | 内容 |
|----|------|
| **空间** | 议事厅墙上铜牌：「同样发明创造只授一项」；小铃「同日实用+发明」 |
| **口述** | 同样发明创造只授一项专利权。例外：同一申请人**同日**既申实用又申发明，先获实用尚未终止且声明放弃实用的，可授发明。两人分别就同样发明创造申请 → 授予**最先申请**的人 |
| **案例锚** | **一发明一权** · **同日双申** · **先申请原则** |
| **关联** | → 2F；申请日 → 28（2F 预告） |

## 4. 寿命与处分台

### 房间 O11｜期限钟楼 · 第 42 条（+43、44 标签）

| 项 | 内容 |
|----|------|
| **空间** | 三口巨钟：发明 **20 年** / 实用新型 **10 年** / 外观 **15 年**，指针都从**申请日**起算；阁楼两架补偿钟：审查延迟补偿、新药补偿 |
| **感官** | 钟摆自申请日起算的错位感（生效看公告、期限看申请日）；年费投币口咔哒（43） |
| **场景钩** | 权利还在不在期；年费断缴终止；新药期限补偿 |
| **30 秒口述** | 发明专利权期限二十年，实用新型十年，外观设计十五年，均自**申请日**起算。发明授权过程不合理延迟可请求期限补偿（申请日起满四年且实审请求日起满三年后授，申请人原因除外）。新药相关发明专利可请求补偿，补偿≤5 年，上市后总有效专利权期限≤14 年 |
| **侧签** | **43** 自授权当年起缴年费；**44** 未缴年费或书面放弃 → 期满前终止并登记公告 |
| **数字桩** | **20 / 10 / 15**；新药补偿桩 **5** 与 **14** |
| **案例锚** | **申请日起算** · **三口期限钟** · **年费终止** |
| **关联** | 过期 → 无 3F 请求权基础；补偿细节 → B0 |
| **商标对照** | 商标 O11 十年续展；专利是「固定期限+年费」，外观已是 15 年 |

### 房间 O12｜转让登记柜 · 第 10 条

| 项 | 内容 |
|----|------|
| **空间** | 双人签字柜：书面合同 +「登记生效」戳；涉外转让另开手续窗 |
