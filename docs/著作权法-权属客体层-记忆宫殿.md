```text
    │

  O12 保护期钟楼(22/23) · O13 原件≠版权厅(20)
  └─ 侧签：许可转让(26/27/29) · 集体管理(8) · 邻接权→3F·R4
    │
    ▼
楼梯 → 3F：接触+实质性相似 · 4F：54 梯子
```

**本层只解决一件事：** 权利是否存在、客体是否适格、权属与处分链条是否清楚、保护期是否还在。  
侵不侵权 → 上 3F；赔不赔 → 上 4F；平台/软件细则 → E1/B0。

**与商标/专利最大差异：** 著作权原则上**自动产生**（不论是否发表），不以登记或公告为生效条件；登记主要是**举证便利**。

## 1. 前厅：权利从哪来、客体是什么

### 房间 O1｜自动产生厅 · 第 2 条

| 项 | 内容 |
|----|------|
| **空间** | 大厅中央没有「核准章」，只有一盏常亮灯：「**不论是否发表**」；墙角小窗写作品登记（O6） |
| **感官** | 灯一亮权就在（创作完成瞬间）；登记章盖不盖灯都亮 |
| **场景钩** | 对方说：没登记、没发表、没版权局证书 → 没权？ |
| **30 秒口述** | 中国公民、法人或非法人组织的作品，**不论是否发表**，依照本法享有著作权。外国人作品依条约/协议或在中国境内出版等规则受保护。权利产生不靠登记公告 |
| **要件** | ① 符合作品条件（→O2） ② 自动保护 ③ 发表不是产生条件 ④ 涉外看条约/首次出版 |
| **案例锚** | **自动产生** · **不论是否发表** · **登记非生效** |
| **关联** | → O2、O6；→ 3F·P1 |
| **商标对照** | 商标 O1＝核准生权；著作权 O1＝创作完成即有 |
| **专利对照** | 专利 O1＝公告生效；著作权无公告门槛 |

### 房间 O2｜作品类型仓 · 第 3 条（↔ 3F·P1）

| 项 | 内容 |
|----|------|
| **空间** | 货架：文字/口述/音乐戏剧舞蹈/美术建筑/摄影/视听/图形模型/计算机软件/其他智力成果；门楣「独创性 + 一定形式表现」 |
| **感官** | 独创性灯亮才开门；纯思想区贴封条 |
| **场景钩** | 商业模式、游戏规则、短句口号算不算作品？ |
| **30 秒口述** | 作品＝文学艺术科学领域内具有**独创性**并能以**一定形式表现**的智力成果；类型清单到软件与其他智力成果。思想、操作方法、单纯事实本身不是本仓货 |
| **要件** | ① 独创性 ② 可感知形式 ③ 对号入座类型 ④ 思想表达二分细拆→3F·S4 |
| **案例锚** | **独创性** · **形式表现** · **类型分流** |
| **关联** | → O5；↔ 3F·P1；视听细拆→O9；**游戏规则 vs 表达 → B16** |
| **侧柜 O2b｜不适用 · 第 5 条** | 法律文件及官方译文、单纯事实消息、历法通用数表公式等不适用本法 |

### 房间 O3｜权利人名单 · 第 9 条

| 项 | 内容 |
|----|------|
| **空间** | 两列名牌：左「作者」；右「其他依法享有著作权的自然人/法人/非法人组织」 |
| **感官** | 名牌可摘换（转让、继承、职务特殊归属后换牌） |
| **场景钩** | 原告是受让人、继承人、单位，还是挂名作者？ |
| **30 秒口述** | 著作权人包括：**作者**；以及其他依照本法享有著作权的自然人、法人或非法人组织（受让、继承、职务特殊归属、视听制作者等） |
| **案例锚** | **作者** · **其他权利人** · **先核主体** |
| **关联** | → O4、O5 |

### 房间 O4｜权利束展厅 · 第 10 条（细拆 ↔ 3F·P2）

| 项 | 内容 |
|----|------|
| **空间** | 一束绳：人身权四股（发表/署名/修改/保护作品完整）+ 财产权多股（复制发行出租展览表演放映广播信息网络传播摄制改编翻译汇编等） |
| **感官** | 人身股烫手难转让；财产股可剪段许可转让 |
| **场景钩** | 只转复制权？信息网络传播权单独卖？署名能不能卖？ |
| **30 秒口述** | 著作权是一束权。人身权四项；财产权多项，可许可或转让并获酬。起诉时先勾被侵害的具体权项——细拆行为与勾选在 3F·P2 |
| **案例锚** | **权利束** · **人身财产分轨** · **先勾权项** |
| **关联** | ↔ 3F·P2；许可转让侧签 26/27/29 |

## 2. 权属走廊：到底是谁的

### 房间 O5｜作者默认厅 · 第 11 条

| 项 | 内容 |
|----|------|
| **空间** | 默认大门写「著作权属于作者」；侧门「法人视为作者」三条件灯：主持 / 代表意志 / 责任 |
| **感官** | 创作自然人指纹印；法人章沉 |
| **场景钩** | 单位说「我们是作者」；个人说「我写的」 |
| **30 秒口述** | 著作权属于作者，另有规定除外。创作作品的**自然人**是作者。由法人或非法人组织**主持**、代表其**意志**创作、并由其**承担责任**的作品，该组织**视为作者** |
| **要件** | ① 默认归作者 ② 自然人创作＝作者 ③ 视为作者三要件 ④ 另有规定→O7/O8/O9 |
| **案例锚** | **作者默认** · **视为作者** · **主持意志责任** |
| **数字桩** | 桩 **1**＝作者红印默认 |
| **关联** | → O6、O7；≠ O7 一般职务（作者仍是个人） |
| **商标/专利对照** | 商标注册人、专利权人靠登记公告；这里靠「谁创作/谁视为作者」 |

### 房间 O6｜署名推定窗 · 第 12 条

| 项 | 内容 |
|----|------|
| **空间** | 玻璃窗贴作品封面署名；窗下抽屉「相反证明」；侧柜「作品登记」章 |
| **感官** | 署名贴纸黏手；反证抽屉沉 |
| **场景钩** | 封面署了公司名；被告否认原告是作者 |
| **30 秒口述** | 在作品上署名的自然人、法人或非法人组织为作者，且该作品上存在相应权利，**但有相反证明的除外**。作者等可向登记机构办理作品登记；邻接权参照适用署名推定与登记规则 |
| **要件** | ① 署名推定 ② 可反证推翻 ③ 登记便利举证非生效 ④ 邻接权参照 |
| **案例锚** | **署名推定** · **相反证明** · **登记举证** |
| **关联** | ↔ O1、O5；诉讼举证常先亮署名+登记 |

### 房间 O7｜职务作品厅 · 第 18 条——本层易混核心

| 项 | 内容 |
|----|------|
| **空间** | 两间工位：左「一般职务」作者享有著作权、单位优先使用；右「特殊职务」作者仅署名、其余归单位 |
| **感官** | 左工位闹钟响两年（完成两年内限制许可）；右工位物质技术条件轰鸣（软件/设计图） |
| **场景钩** | 员工离职带走代码、设计图、稿件起诉单位或被诉 |
| **30 秒口述** | **一般职务作品**：为完成工作任务创作 → 著作权归**作者**，单位在业务范围内**优先使用**；完成**两年内**未经单位同意，作者不得许可第三人以与单位相同方式使用。**特殊职务作品**（作者仅署名、其余归单位）：①主要利用单位物质技术条件创作且单位承担责任的工程设计图、产品设计图、地图、示意图、**计算机软件**等；②报社期刊社通讯社广播电台电视台工作人员创作的职务作品；③法律行政法规规定或合同约定归单位的 |
| **要件** | ① 先分一般/特殊 ② 一般≠视为作者 ③ 两年限制 ④ 软件常落特殊① |
| **案例锚** | **一般职务** · **特殊职务** · **软件归单位** · **两年限制** |
| **数字桩** | 桩 **2**＝两年限制；桩 **0**＝特殊情形作者财产权落空（仅留署名） |
| **关联** | ≠ O5 视为作者；↔ O8 委托；软件细则→E1 |
| **专利对照** | ≈ 专利职务发明(6)，但著作权默认更常「权归作者」 |

### 房间 O8｜委托作品厅 · 第 19 条

| 项 | 内容 |
|----|------|
| **空间** | 合同桌：甲方委托人、乙方受托人；桌角大字「**未约定＝归受托人**」 |
| **感官** | 合同纸涩；空白栏刺眼 |
| **场景钩** | 定制 Logo、外包文案、委托拍照，事后争版权 |
| **30 秒口述** | 受委托创作的作品，归属由委托人和受托人**合同约定**；合同未明确约定或没有订立合同的，著作权属于**受托人** |
| **要件** | ① 约定优先 ② 默认归受托人 ③ 付钱≠自动得版权 |
| **案例锚** | **约定优先** · **默认受托人** · **付款≠得权** |
| **关联** | ↔ O7；→ O13（原件买卖另论） |
| **商标对照** | 委托设计商标图样常另签转让；著作权默认不跟钱走 |

### 房间 O9｜视听作品厅 · 第 17 条

| 项 | 内容 |
|----|------|
| **空间** | 摄影棚：中央「制作者」椅；侧排编剧导演摄影作词作曲署名牌；后墙「其他视听可约定」；角落可单独使用的剧本/音乐门 |
| **感官** | 场记板啪；合同页翻动 |
| **场景钩** | 短视频、网剧、电影、综艺：谁起诉？编剧能不能单独告剧本被抄？ |
| **30 秒口述** | **电影、电视剧**：著作权由**制作者**享有；编剧、导演、摄影、作词、作曲等作者享有**署名权**，并有权按合同获酬。其他视听作品：归属**当事人约定**；无约定或约定不明 → 由**制作者**享有，作者享有署名权和获酬权。剧本、音乐等可单独使用的作品，作者有权**单独行使**其著作权 |
| **要件** | ① 影视默认制作者 ② 主创署名+获酬 ③ 其他视听约定优先 ④ 可分割作品单独行权 |
| **案例锚** | **制作者** · **主创署名** · **单独作品** |
| **关联** | ↔ O2 视听类型；↔ 3F·R3/R4；保护期→O12 |
| **专利对照** | 无直接对应；记「整片归制作者，零件可单行」 |

### 房间 O10｜合作作品厅 · 第 14 条

| 项 | 内容 |
|----|------|
| **空间** | 双人桌；门口拦杆写「没参加创作＝不是合作作者」；墙上议事规则：协商一致；不能协商不得乱转专有许可 |
| **感官** | 两支笔同响；收益分账计算器 |
| **场景钩** | 挂名共同作者、合作破裂一方独家授权 |
| **30 秒口述** | 两人以上合作创作，著作权由合作作者共同享有；没参加创作的不能成为合作作者。行使原则上协商一致；不能协商又无正当理由的，一方不得阻止他方行使除转让、许可他人专有使用、出质以外的其他权利，但收益应合理分配。可分割使用的，各部分作者可单独享有，但不得侵犯整体著作权 |
| **案例锚** | **共同享有** · **非创作非作者** · **可分割单独** |
| **关联** | → O11；保护期按最后死亡作者→O12 |

### 房间 O11｜演绎·汇编廊 · 第 13、15、16 条

| 项 | 内容 |
|----|------|
| **空间** | 三连廊：改编翻译注释整理(13) → 汇编选择编排(15) → 使用演绎汇编作品还要过原作许可(16) |
| **感官** | 原作幽灵跟在身后（双重许可） |
| **场景钩** | 翻译小说出版、汇编数据库、改编游戏 |
| **30 秒口述** | 演绎作品著作权归演绎人，但行使不得侵犯原作。汇编作品（对内容选择或编排体现独创性）归汇编人，亦不得侵犯原作。出版、演出、制作录音录像制品使用演绎/汇编作品时，须取得**该作品**与**原作品**著作权人许可并付酬 |
| **案例锚** | **演绎新作** · **汇编独创** · **双重许可** |
| **关联** | ↔ 3F 比对仍看表达；→ O8 委托翻译常见 |
| **数字桩** | 桩 **2**＝双重许可对撞 |

## 3. 寿命与处分台

### 房间 O12｜保护期钟楼 · 第 22、23 条（↔ 3F·S5）

| 项 | 内容 |
|----|------|
| **空间** | 三面钟：人身权「不受限制」；自然人「终生+死后五十年」；法人/职务特殊/视听「五十年档」 |
| **感官** | 12 月 31 日钟声；沙漏见底＝公有领域 |
| **场景钩** | 古典作品、老照片、未发表遗作、视听片 |
| **30 秒口述** | **署名权、修改权、保护作品完整权**保护期不受限制。自然人作品：发表权与财产权一般为作者终生及其死亡后五十年（合作作品按最后死亡作者）。法人作品及特殊职务作品、视听作品：发表权自创作完成起算五十年档；财产权自首次发表起算五十年档，创作完成后五十年未发表的不再保护。截止日均为相应第五十年的 **12 月 31 日** |
| **案例锚** | **人身权无限期** · **终生加五十年** · **视听五十年** · **年底截止** |
| **数字桩** | 桩 **5**＝五十年；桩 **0**＝过期进公有领域 |
| **关联** | ↔ 3F·S5；详细起算争议→B0 |

### 房间 O13｜原件≠版权厅 · 第 20 条

| 项 | 内容 |
|----|------|
| **空间** | 画廊：画框卖掉，版权吊牌仍挂作者名；例外小灯「美术摄影原件展览权随原件」 |
| **感官** | 画框易手「咔」；版权吊牌不动 |
| **场景钩** | 买了原画就以为可以出画册、上架 NFT？ |
| **30 秒口述** | 作品**原件所有权**转移，**不改变**著作权归属；但美术、摄影作品原件的**展览权**由原件所有人享有。作者将未发表的美术、摄影作品原件转让后，受让人展览该原件不构成对发表权的侵犯 |
| **案例锚** | **物债分离** · **展览权例外** · **买画≠买版权** |
| **关联** | ↔ O8；处分财产权仍走 26/27 |
