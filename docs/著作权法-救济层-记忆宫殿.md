# 著作权法 · 4F 救济层｜完整记忆宫殿房间表

> 依据：《中华人民共和国著作权法》（2020 年修正）  
> 定位：律所大楼「著作权馆」四楼——认定侵权之后，谈路径、谈停、谈赔、谈诉前与销毁  
> 上楼入口：3F 走廊尽头楼梯（见 `著作权法-侵权认定层-记忆宫殿.md`）  
> 对照：商标馆 4F / 专利馆 4F  
> 数据：`docs/data/copyright-palace.json`

## 0. 本层总图

```text
3F 楼梯上来
    │
    ▼
接待台【路径分流】60：调解 / 仲裁 / 起诉
    │         └─ 行政蓝灯：53 损害公益时主管著作权部门查处（+55 职权）
    │
    ├── 左翼【行政查处车间】53 行政段 + 55
    │         责令停止·警告·没收销毁·五倍/二十五万罚款·刑事接口
    │
    ├── 中厅【赔偿殿】54
    │         损失或违法所得 → 使用费 → 惩罚阁楼(故意×1～5)
    │         → 法定区间(500元～500万) · 举证抽屉 · 销毁后室
    │
    ├── 右翼【举证/来源】
    │         F5 授权·来源·许可抗辩窗 59 · F4 时效钟（民事一般规则）
    │
    ├── 急诊通道【诉前】56 财产保全+行为禁令 → 57 证据保全
    │
    └── 侧柜【法院没收】58 · 红灯【刑事】53 末句
```

**本层只解决一件事：** 走哪条救济路径、停不停、罚不罚、赔多少、来不来得及保全、毁不毁。  
侵权能不能成立 → 回 3F。

**与专利对照点：** 著作权 **53 条**在损害公共利益时可直接走「五倍/二十五万」行政罚款（更像商标行政），不是专利那种「普通侵权行政少罚款、假冒另开厅」。

## 1. 接待台｜第 60 条（+ 行政入口）

### 房间 F1｜分流接待台 · 第 60 条

| 项 | 内容 |
|----|------|
| **空间** | 楼梯口接待台，三盏灯：**调解黄 / 仲裁橙 / 法院红**；侧壁加一盏 **行政蓝**（损害公益→53） |
| **感官** | 电话忙音；柜台笃笃敲击；蓝灯偶闪（盗版规模大、危害广） |
| **场景钩** | 3F 已认定侵权，当事人问：调解？仲裁？起诉？要不要投诉版权局？ |
| **30 秒口述** | 著作权纠纷可以**调解**；有书面仲裁协议或合同仲裁条款的可申请**仲裁**；没有的，可直接向法院**起诉**。同时损害公共利益的侵权行为，还可由主管著作权的部门依 **53** 查处（停、没、罚，甚至移送刑事） |
| **要件** | ① 民事主路径：调解/仲裁/诉讼 ② 行政路径看是否损害公益+53 情形 ③ 赔偿谈判常最终回法院 |
| **案例锚** | **调解仲裁诉讼** · **行政并行可能** · **先选路径** |
| **关联** | → F2、F3；← 3F·R3 |
| **商标对照** | ≈ 商标 F1（60 三灯）；著作权多「仲裁」入口，行政挂 53 而非单独 60② |
| **专利对照** | 专利行政处理写在 65；著作权行政罚则嵌在 53 |

| **专利对照** | 专利普通侵权行政≠这套罚则；著作权 53 行为本身就可接行政 |

## 3. 中厅｜第 54 条（赔偿殿）——本层核心

> 与专利 71 同构：**损失或者违法所得**并列优先（不是商标严格「先损后利」）。法定下限是 **500 元**（不是专利 3 万）。

```text
站①实际损失 ∥ 站②违法所得
    → 站③权利使用费
    → 站④惩罚阁楼(故意+情节严重 → 1～5倍)
    → 站⑤法定区间柜台(500元～500万)
         旁门：举证抽屉 · 销毁后室
```

### 房间 F3-①／②｜损失柜 ∥ 违法所得柜 · 第 54 条第一款前段

| 项 | 内容 |
|----|------|
| **空间** | 两柜并排：左「权利人损失」，右「侵权人违法所得」；门楣「**或者**」 |
| **感官** | 两台计算器滴答 |
| **口述** | 赔偿按权利人因被侵权所受**实际损失**或者侵权人的**违法所得**确定 |
| **桩** | 桩 **1**＝损失；桩 **2**＝违法所得；「或者」灯提醒并列优先 |
| **商标/专利对照** | 同专利「或」；不同于商标「损→利」强制顺序 |

### 房间 F3-③｜使用费柜台 · 第 54 条第一款中段

| 项 | 内容 |
|----|------|
| **空间** | 第三柜：许可使用费合同复印件 |
| **口述** | 损失与违法所得难以计算 → **参照该权利使用费**给予赔偿 |
| **桩** | 桩 **3**＝第三站 |

### 房间 F3-④｜惩罚性阁楼 · 第 54 条第一款惩罚段

| 项 | 内容 |
|----|------|
| **空间** | 窄楼梯通阁楼，门牌「**故意** × **情节严重**」 |
| **感官** | 阁楼热风（故意） |
| **口述** | 对**故意**侵犯著作权或邻接权且**情节严重**的，可在按上述方法确定数额的**一倍以上五倍以下**确定赔偿 |
| **要件** | ① 已有基数方法 ② 故意 ③ 情节严重 ④ 倍数 1～5 |
| **数字桩** | 桩 **5**＝五倍；与行政 53「五倍」同桩不同场景 |
| **案例锚** | **故意严重** · **惩罚性** · **一至五倍** |
| **对照** | 用词「故意」（同专利）；商标写「恶意」 |

### 房间 F3-⑤｜法定区间柜台 · 第 54 条第二款

| 项 | 内容 |
|----|------|
| **空间** | 尽端柜台铜牌：「**五百元以上**」「**五百万元以下**」 |
| **感官** | 「五百元」小字要凑近看（下限很低） |
| **口述** | 损失、违法所得、使用费均难计算 → 法院依情节判给**五百元以上五百万元以下** |
| **数字桩** | 桩 **5**＝五百万封顶；下限记「五百元起」别记成专利三万 |
| **案例锚** | **法定赔偿** · **五百元起** · **五百万封顶** |

### 房间 F3-旁｜举证抽屉 · 第 54 条第四款

| 项 | 内容 |
|----|------|
| **空间** | 违法所得柜旁铁抽屉 |
| **口述** | 权利人已尽必要举证责任，账簿资料主要由侵权人掌握 → 可责令提供；不提供或虚假提供 → 可参考权利人主张和证据定赔 |
| **案例锚** | **举证妨碍** · **账簿令** |

### 房间 F3-后｜销毁后室 · 第 54 条第五款

| 项 | 内容 |
|----|------|
| **空间** | 后门车间：侵权复制品进销毁口；模具工具进熔炉；告示「不予补偿」 |
| **口述** | 应权利人请求：侵权复制品除特殊情况外**责令销毁**；主要用于制造的材料、工具、设备责令销毁且**不予补偿**；特殊情况可禁入商业渠道且不补偿 |
| **案例锚** | **销毁不计补偿** · **禁入商业渠道** |
| **关联** | ↔ F2 行政销毁；↔ 58 法院没收 |

### 房间 F3-贴｜合理开支 · 第 54 条第三款

| 项 | 内容 |
|----|------|
| **口述** | 赔偿数额**还应当包括**权利人为制止侵权支付的合理开支 |

**54 条闭眼口诀：**  
损或违法所得 → 使用费 →（故意严重×1～5）→ **500 元～500 万**；加合理开支；账簿可责令；复制品原则上销毁。

## 4. 右翼｜举证窗 + 时效钟

### 房间 F5｜授权·来源·抗辩窗 · 第 59 条（↔ 3F·S3）

| 项 | 内容 |
|----|------|
| **空间** | 两扇窗：左「出版制作＝合法授权」；右「发行出租＝合法来源」；窗下抽屉「诉讼中主张免责须举证许可或法定不经许可情形」 |
| **感官** | 授权合同涩手；进货单油墨 |
| **30 秒口述** | 出版者、制作者不能证明合法授权，发行者或软件/视听/音像复制品出租者不能证明合法来源的，应承担法律责任。诉讼中被告主张不承担责任的，应举证已获许可，或具有本法规定的不经许可可使用的情形（如合理使用） |
| **要点** | ① 流通环节举证 ② 免抗辩也要举证 ③ 与 3F·S3 同条，4F 当「赔不赔/责不责」入口 |
| **案例锚** | **合法授权** · **合法来源** · **抗辩举证** |
| **商标/专利对照** | ≈ 商标 64② / 专利 77；著作权 59 第二款把「许可/法定例外」举证写得更硬 |

### 房间 F4｜时效钟房（民事一般规则）

| 项 | 内容 |
|----|------|
| **空间** | 钟楼未单独立著作权法条号，挂牌「民法典三年」；沙漏旁便签「持续侵权注意起算」 |
| **感官** | 钟摆沉重 |
| **场景钩** | 多年前的盗版、持续在线传播 |
| **30 秒口述** | 著作权法未单列类似专利 74 的侵权时效专条；诉讼时效一般适用**民法典**关于三年时效等规则，从知道或应当知道权利受损及义务人之日起算。持续提供侵权内容时，起算与赔偿期间常成争点（细则 B0） |
| **案例锚** | **三年时效** · **知权利受损与义务人** · **持续侵权** |
| **对照** | 专利有 74 本法钟；著作权先记「回民法典」 |

## 10. 通勤闭眼脚本（约 8～10 分钟）

1. **F1**（30 秒）：调解、仲裁、起诉，要不要亮行政蓝灯  
2. **F2**（1.5 分）：损害公益？5 万/五倍/25 万  
3. **F3**（4 分）：损或所得→使用费→故意×1～5→500 元～500 万；开抽屉与销毁后室  
4. **F5+F4**（1.5 分）：授权来源举证？过没过时效？  
5. **F6→F7**（1.5 分）：禁令与证保来不来得及  
6. **F8**（30 秒）：要不要刑事红灯

## 11. 读案五问 · 第 5 问展开

| 子问 | 落点 |
|------|------|
| 停不停？ | 判决停止 / 53 行政责令 / 56 诉前禁令 |
| 罚不罚？ | 53 损害公益 + 罚款阶梯 |
| 赔不赔？ | 先 F5·59；再进 F3·54 |
| 赔多少？ | 54 五站 + 合理开支 + 举证妨碍 |
| 毁不毁？ | 54 销毁 / 53 行政销毁 / 58 没收 |
| 来得及吗？ | 56 / 57 |
| 要不要刑？ | 53 末句 + F8 |

**30 秒假想案：**  
批量网盘传播盗版影视并收费 → 3F 走 R3；4F 开 F2 看公益与罚款，F3 冲故意惩罚性，链接朝不保夕 → F7；发行渠道说不清来源 → F5。
