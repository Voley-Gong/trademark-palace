## 1. 前厅：权利从哪来

### 房间 O1｜注册专用权厅 · 第 3 条

| 项 | 内容 |
|----|------|
| **空间** | 大厅中央立着注册证铜匣，匣盖刻「核准 = 专用权」 |
| **感官** | 铜匣冰凉；启封有「咔」一声（权利产生瞬间） |
| **场景钩** | 对方质疑：未注册有没有专用权？集体/证明商标怎么算？ |
| **30 秒口述** | 经商标局**核准注册**的商标为注册商标（商品/服务/集体/证明）；**注册人享有商标专用权**，受法律保护 |
| **要件** | ① 核准注册 ② 注册人 ③ 专用权受保护 ④ 分清商标类型 |
| **案例锚** | **注册取得** · 专用权起点 · 类型分流 |
| **关联** | → O2；→ 3F·P1（56 边界）；驰名另乘 3F·E1 |
| **数字桩** | 桩 **1**＝独占红印：专用权因注册而生 |

### 房间 O2｜申请入口 · 第 4 条

| 项 | 内容 |
|----|------|
| **空间** | 前厅侧门「申请窗口」，窗口上方红灯写「恶意驳回」 |
| **感官** | 排队嘈杂；红灯刺眼 |
| **场景钩** | 囤积抢注、不以使用为目的的批量申请 |
| **30 秒口述** | 生产经营中需要取得专用权的，应向商标局申请注册。**不以使用为目的的恶意商标注册申请，应当予以驳回**。商品商标规定适用于服务商标 |
| **要件** | ① 申请主体适格 ② 生产经营需要 ③ 恶意不以使用为目的 → 驳回 |
| **案例锚** | **恶意申请** · 使用目的 · 驳回 |
| **关联** | ↔ 第7条诚信；↔ 无效 44（绝对事由，2F/地下室） |

## 2. 客体走廊：标志能不能当商标

### 房间 O3｜标志仓库 · 第 8 条

| 项 | 内容 |
|----|------|
| **空间** | 走廊仓库：文字、图形、字母、数字、立体、颜色组合、声音货架 |
| **感官** | 货架金属碰撞；声音标样在耳机里短促响一声 |
| **场景钩** | 非传统商标、组合标志、是否「能够区别来源」 |
| **30 秒口述** | 任何能将商品/服务与他人区别开的标志（含文字、图形、字母、数字、三维、颜色组合、声音等及组合）均可作为商标申请注册 |
| **要件** | ① 来源区别可能性 ② 法定要素或组合 |
| **案例锚** | **可注册标志** · 来源区别 · 非传统商标 |
| **关联** | → O4 显著性审查 |

### 房间 O4｜显著总纲门 · 第 9 条

| 项 | 内容 |
|----|------|
| **空间** | 双扇门：左门「显著特征」，右门「不得与在先权利冲突」 |
| **感官** | 推门需双手同时用力（两要件并行） |
| **场景钩** | 既像描述性又撞别人字号/著作权 |
| **30 秒口述** | 申请注册的商标应有**显著特征、便于识别**，并**不得与他人在先取得的合法权利相冲突**。注册人有权标明注册标记 |
| **要件** | ① 显著 ② 不与在先权利冲突 |
| **案例锚** | **显著+在先** · 双门槛 |
| **关联** | → O5/O6 绝对事由；→ O9 第32条展开 |

### 房间 O5｜禁用红线柜 · 第 10 条

| 项 | 内容 |
|----|------|
| **空间** | 上锁红柜，柜门列国旗、国徽、官方标志、欺骗性、不良影响、地名等 |
| **感官** | 红漆刺鼻；柜锁冰冷 |
| **场景钩** | 国名国旗近似、欺骗性宣传、行政区划地名 |
| **30 秒口述** | 若干标志**不得作为商标使用**（国名国旗等、官方标志、民族歧视、欺骗性、不良影响等）；县级以上行政区划地名或公众知晓外国地名原则上不得作商标（有例外） |
| **要件** | 落入禁用清单 → 连「使用」都不允许，不只是不给注册 |
| **案例锚** | **禁用标志** · 使用禁令 · 地名例外 |
| **关联** | 与 O6「不得注册」分层记忆 |

### 房间 O6｜禁注三格 · 第 11 条

| 项 | 内容 |
|----|------|
| **空间** | 三格陈列柜：①通用名称 ②直接描述 ③其他缺乏显著；柜底抽屉写「第二含义」 |
| **感官** | 抽屉拉开有使用痕迹的磨砂感（经过使用获得显著） |
| **场景钩** | 「纯描述」「通用名」「用出来了显著性」 |
| **30 秒口述** | 仅通用名称/图形/型号，或仅直接表示质量原料功能等，或其他缺乏显著特征的，**不得注册**；但经过使用取得显著特征并便于识别的，可以注册 |
| **要件** | ① 三类缺乏显著 ② 使用取得显著的例外 |
| **案例锚** | **缺乏显著** · **第二含义** · 仍可注 |
| **关联** | ↔ 3F·S2 正当使用（59）；显著弱则侵权比对也吃亏 |

### 房间 O7｜立体禁注角 · 第 12 条

| 项 | 内容 |
|----|------|
| **空间** | 展台放三个「不能注」的立体模型：性质形状 / 技术效果 / 实质价值 |
| **感官** | 模型棱角硌手 |
| **场景钩** | 产品外形、为技术效果必需的形状、使商品具有实质价值的形状 |
| **30 秒口述** | 三维标志申请：仅由商品自身性质产生的形状、为获得技术效果所需形状、或使商品具有实质性价值的形状，**不得注册** |
| **案例锚** | **功能性排除** · 立体商标 |
| **关联** | ↔ 3F·59② 正当使用形状 |

## 3. 权属冲突侧廊

### 房间 O8｜抢注暗室 · 第 15 条

| 项 | 内容 |
|----|------|
| **空间** | 昏暗签约室：代理人名片与注册证叠在一起 |
| **感官** | 纸张摩挲；门锁「咔哒」像背叛 |
| **场景钩** | 代理/代表抢注；合同业务往来中明知未注册商标却抢注 |
| **30 秒口述** | ① 代理人/代表人以自己名义将被代理人商标注册，被异议 → 不予注册并禁止使用；② 同种/类似商品上，申请人因合同业务等关系明知他人在先未注册商标存在而申请，被异议 → 不予注册 |
| **案例锚** | **代理抢注** · 关系人明知 · 禁止使用 |
| **关联** | ↔ O9；程序上常进异议/无效（2F） |

### 房间 O9｜在先权利墙 · 第 32 条

| 项 | 内容 |
|----|------|
| **空间** | 整面石墙嵌着「著作权、姓名权、字号、外观设计…」铭牌；墙角小龛写「有一定影响」 |
| **感官** | 石墙凉意；铭牌叮当作响（多种在先权） |
| **场景钩** | 撞著作权、姓名、字号；或抢注别人已使用并有一定影响的未注册商标 |
| **30 秒口述** | 申请商标注册**不得损害他人现有的在先权利**，也不得以不正当手段**抢先注册**他人已经使用并有**一定影响**的商标 |
| **要件** | ① 在先权利存在 ② 或：在先使用+一定影响+不正当抢注 |
| **案例锚** | **在先权利** · **抢注一定影响** · 不正当手段 |
| **关联** | ↔ O4；↔ 3F·59③ 先用权抗辩（侵权侧）；↔ 无效 45 |

### 房间 O10｜申请在先钟 · 第 31 条

| 项 | 内容 |
|----|------|
| **空间** | 走廊挂钟，指针停在「申请日」；旁边小铃「同日→使用在先」 |
| **感官** | 钟摆声；同日铃声叮 |
| **场景钩** | 两个近似商标撞车，谁先申请 |
| **30 秒口述** | 同一种或类似商品上，相同或近似商标多人申请 → **初步审定并公告申请在先的**；**同一天申请** → 初步审定并公告**使用在先的**，驳回其他人 |
| **案例锚** | **申请在先** · 同日使用在先 |
| **数字桩** | 与优先权「六个月」区分：那是国际/展览优先权（25/26，挂 2F 预告） |
| **关联** | → 2F 审查程序 |

## 4. 寿命与处分台

### 房间 O11｜十年续展台 · 第 39、40 条

| 项 | 内容 |
|----|------|
| **空间** | 服务台日历巨大：有效期十年；柜台左侧「期满前12个月」，右侧「宽展6个月」 |
| **感官** | 翻日历沙沙响；宽展铃轻响 |
| **场景钩** | 权利还在不在、过期未续展、宽展期内补办 |
| **30 秒口述** | 注册商标有效期**十年**，自核准注册之日起算。期满需继续使用：期满前**十二个月**办理续展；未办可有**六个月**宽展；每次续展十年，自上一届期满次日起算；期满未续展 → 注销 |
| **数字桩** | **10 年**＝有效期/续展期；**12 个月**办理窗；**6 个月**宽展（可用桩 6 的形象自建：六面骰＝宽展） |
| **案例锚** | **十年有效** · **续展宽展** · 注销 |
| **关联** | 过期权利影响 3F 请求权基础 |

### 房间 O12｜转让柜 · 第 42 条

| 项 | 内容 |
|----|------|
| **空间** | 双人签字柜：转让协议 +「一并转让」捆绳 |
| **感官** | 捆绳勒手（近似商标要一并转） |
| **场景钩** | 只转主标不转近似标；转让是否导致混淆 |
| **30 秒口述** | 转让须签协议并共同申请；受让人保证商品质量；同一种商品上的近似商标或类似商品上的相同/近似商标应**一并转让**；易混淆或不良影响的不予核准；核准公告后，受让人自**公告之日**起享有专用权 |
| **案例锚** | **一并转让** · 公告日起权属 · 质量保证 |
| **关联** | ↔ O13 许可（处分方式不同） |

### 房间 O13｜许可窗 · 第 43 条

| 项 | 内容 |
|----|------|
| **空间** | 玻璃窗后挂许可合同；窗贴「备案对抗」便签 |
| **感官** | 玻璃反光；便签胶粘手 |
| **场景钩** | 被许可人维权资格、未备案、商品未标名称产地 |
| **30 秒口述** | 可签订使用许可合同；许可人监督质量，被许可人保证质量；商品上须标明被许可人名称和产地；许可应报商标局备案公告；**未经备案不得对抗善意第三人** |
| **案例锚** | **许可备案** · 对抗善意第三人 · 质量与标注 |
| **关联** | 利害关系人起诉资格常连司法解释（地下室） |

## 8. 读案五问 · 本层落点

| 五问 | 1F 落点 |
|------|---------|
| 1. 请求权基础？ | O1 注册专用权；未注册/字号可能改道反法或 32 思路 |
| 2. 权利客体/权属？ | O3–O7 适格；O8–O10 冲突；O11 是否仍有效；O12/O13 谁有权主张 |
| 3–5 | 上 3F / 4F |

**30 秒假想案：**  
对方主张你的注册标缺乏显著且抢了他们在先字号 → O6 + O9；若你方过期未续展 → 先开 O11；许可他人生产要起诉 → O13 备案与利害关系。
