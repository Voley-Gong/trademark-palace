## 0. 本层总图

```text
电梯口（争议定性：发明/实用新型/外观？有效专利？）
    │
    ▼
前厅【保护范围】P1 第64条 ──►【实施行为】P2 第11条
    │
    ▼
主走廊（落入判定）
  R1 字面全面覆盖 → R2 等同镜廊 → R3 方法专利举证闸
  → R4 外观设计比对厅
    │
    ├── 抗辩侧廊
    │     S1 现有技术/设计(67) · S2 先用权(75②)
    │     S3 权利用尽(75①) · S4 其他不视为侵权(75③④⑤)
    │     S5 合法来源免赔(77)
    ├── 口径侧窗【禁止反悔/捐献】W1（解释补丁，连未来 B0）
    └── 电梯【无效/评价报告】E1（预告 2F · 第66条）
    │
    ▼
楼梯预告 → 4F 救济（65 路径 / 71 赔偿 / 72 禁令 / 73 证保）
```

**本层只解决一件事：** 被控技术方案（或设计）是否落入保护范围、实施了哪些行为、有哪些法定抗辩。  
权属与权利要求怎么写 → 1F；无效宣告 → 2F；赔多少 → 4F。

## 1. 前厅

### 房间 P1｜保护范围门 · 第 64 条

| 项 | 内容 |
|----|------|
| **空间** | 铁门两侧：左挂「权利要求书」，右挂「外观图片/照片」；说明书是门后的说明书灯 |
| **感官** | 纸页重量感；图纸荧光 |
| **场景钩** | 保护范围争到说明书实施例、外观争简要说明 |
| **30 秒口述** | 发明/实用新型：保护范围以**权利要求内容**为准，说明书及附图可以用于**解释**权利要求。外观设计：以图片或照片中的该产品外观设计为准，简要说明可以用于解释 |
| **要件** | ① 有效专利 ② 确定权利要求（或外观视图）③ 解释不得脱权利要求另立范围 |
| **案例锚** | **权利要求为准** · **解释而非替换** · 外观以视图为准 |
| **关联** | → P2、R1、R2、R4 |
| **商标对照** | ≈ 商标 P1（56 核准×核定）：都是「权利边界先钉死」 |

### 房间 P2｜实施行为门 · 第 11 条

| 项 | 内容 |
|----|------|
| **空间** | 五扇小门（发明实用新型）：制造 / 使用 / 许诺销售 / 销售 / 进口；方法专利另开「使用方法+延伸产品」暗门；外观少一扇「使用」 |
| **感官** | 门轴各自声响不同；「生产经营目的」牌烫手 |
| **场景钩** | 广告许诺销售、进口、使用方法专利产品 |
| **30 秒口述** | 发明实用新型：未经许可，不得为**生产经营目的**制造、使用、许诺销售、销售、进口专利产品，或使用专利方法及使用、许诺销售、销售、进口依该方法直接获得的产品。外观：制造、许诺销售、销售、进口外观设计专利产品 |
| **要件** | ① 生产经营目的 ② 未经许可 ③ 落入第11条行为类型 ④ 对象落入保护范围（回 P1/R 廊） |
| **案例锚** | **五类实施** · **许诺销售** · **方法延伸** |
| **关联** | → R 廊；合法来源免赔只及部分行为（S5） |
| **商标对照** | ≈ 商标 P2+57：一边是「用标」，一边是「实施专利」 |

## 2. 主走廊：落入判定

### 房间 R1｜字面铁门 · 全面覆盖（解释口径 + 第 64 条）

| 项 | 内容 |
|----|------|
| **空间** | 厚门上每个技术特征一把锁，钥匙孔要对齐被控方案 |
| **感官** | 锁扣金属咔哒（特征一一对应） |
| **场景钩** | 被控产品是否具备权利要求全部技术特征 |
| **30 秒口述** | 字面侵权（全面覆盖）：被控技术方案包含与权利要求记载的**全部技术特征相同**的特征 → 落入保护范围 |
| **要件** | ① 划分技术特征 ② 逐一比对 ③ 全部覆盖且相同 |
| **案例锚** | **全面覆盖** · **特征比对** · 字面落入 |
| **商标对照** | ≈ 商标 R1（57① 双重相同）：最硬的「相同」通道 |
| **数字桩** | 桩 **1**＝字面独占红线 |

### 房间 R2｜等同镜廊（解释口径）

| 项 | 内容 |
|----|------|
| **空间** | 哈哈镜：特征「基本相同的手段、实现基本相同的功能、达到基本相同的效果」，且所属领域普通技术人员无需创造性劳动即可联想到 |
| **感官** | 镜面变形仍可认；耳语「容易联想到」 |
| **场景钩** | 字面差一个特征，但是否构成等同替换 |
| **30 秒口述** | 等同侵权：与权利要求特征相比，以基本相同手段、实现基本相同功能、达到基本相同效果，并且本领域普通技术人员无需创造性劳动即可联想到的特征 → 视为等同，落入保护范围（受禁止反悔、捐献等限制，见 W1） |
| **要件** | ① 手段/功能/效果基本相同 ② 显而易见联想 ③ 不与禁止反悔等冲突 |
| **案例锚** | **等同三基本** · **联想到** · 限制规则 |
| **商标对照** | ≈ 商标 R2（57② 近似+混淆）：都是「不像一模一样，但法律仍可能认」 |
| **数字桩** | 桩 **2**＝双方特征对照 |

### 房间 R3｜方法专利举证闸 · 第 66 条第一款

| 项 | 内容 |
|----|------|
| **空间** | 闸机标「新产品制造方法」：举证责任倒置提示灯 |
| **感官** | 红灯亮起「你证明方法不同」 |
| **场景钩** | 新产品方法专利，被告生产同样产品 |
| **30 秒口述** | 专利侵权纠纷涉及**新产品制造方法**的发明专利，制造同样产品的单位或个人应当提供其产品制造方法**不同于**专利方法的证明 |
| **要件** | ① 新产品 ② 方法发明专利 ③ 同样产品 → 被告证不同方法 |
| **案例锚** | **方法专利** · **举证责任倒置** · 新产品 |
| **关联** | ↔ P2 方法实施；证据细节进未来 B0 |
| **商标对照** | 商标少有「方法倒置」；记作专利特有闸机 |

### 房间 R4｜外观比对厅 · 第 64 条第二款 + 解释

| 项 | 内容 |
|----|------|
| **空间** | 展柜灯光下产品与授权图片并排；「整体观察、综合判断」标语 |
| **感官** | 灯光均匀；转盘看设计空间 |
| **场景钩** | 外观设计专利侵权，是否相同或近似 |
| **30 秒口述** | 外观保护范围以图片或照片为准；侵权比对通常整体观察、综合判断，以一般消费者知识水平和认知能力，看是否相同或近似（设计空间等作补丁） |
| **案例锚** | **整体观察** · **一般消费者** · 视图为准 |
| **商标对照** | ≈ 商标近似比对，但参照主体是「一般消费者」对设计，不是商标「相关公众对来源」 |

## 3. 抗辩侧廊

### 房间 S1｜现有技术/设计盾 · 第 67 条

| 项 | 内容 |
|----|------|
| **空间** | 盾牌刻「现有技术」「现有设计」；一盾挡下侵权矛 |
| **感官** | 盾面冰凉；文献纸张味 |
| **场景钩** | 被告主张我用的是申请日前已公开的技术/设计 |
| **30 秒口述** | 被控侵权人有证据证明其实施的技术或设计属于**现有技术或现有设计**的，**不构成**侵犯专利权 |
| **要件** | ① 现有技术/设计证据 ② 被控方案与之相同（或依法构成现有技术抗辩的比对标准） |
| **案例锚** | **现有技术抗辩** · **不构成侵权** · 公开证据 |
| **商标对照** | 类似「我用的是公共领域」，但商标侧更多走正当使用/先用；专利此盾极常用 |

### 房间 S2｜先用权作坊 · 第 75 条第二项

| 项 | 内容 |
|----|------|
| **空间** | 申请日之前的旧作坊：已制造相同产品/使用相同方法，或已作好必要准备；墙上线「仅原有范围」 |
| **感官** | 机油味；日历停在申请日前 |
| **场景钩** | 自己申请日前已投入，被诉后主张先用 |
| **30 秒口述** | 申请日前已制造相同产品、使用相同方法，或已作好制造、使用的必要准备，并且**仅在原有范围内**继续制造、使用的，不视为侵权 |
| **要件** | ① 申请日前 ② 已制造/使用或必要准备 ③ 原有范围继续 |
| **案例锚** | **先用权** · **必要准备** · **原有范围** |
| **商标对照** | ≈ 商标 59③ 先用权窗（更像：时间在先 + 范围限制） |

### 房间 S3｜权利用尽闸 · 第 75 条第一项

| 项 | 内容 |
|----|------|
| **空间** | 售出后的商品通道闸机抬起：使用、许诺销售、销售、进口放行 |
| **感官** | 闸机抬杆声 |
| **场景钩** | 正品售出后的转售、进口平行话题（注意国境与许可范围争议） |
| **30 秒口述** | 专利产品或依专利方法直接获得的产品，由专利权人或其许可的单位、个人**售出后**，使用、许诺销售、销售、进口该产品的，不视为侵权 |
| **案例锚** | **权利用尽** · **售出后** · 流转自由 |
| **商标对照** | 商标也有真品流转思路，但法条结构不同；此处单记专利 75① |

### 房间 S4｜其他不视为侵权舱 · 第 75 条第三～五项

| 项 | 内容 |
|----|------|
| **空间** | 三个舱：临时过境交通工具 / 科研实验 / 药品医疗器械行政审批（Bolar） |
| **感官** | 引擎声、实验室玻璃声、药盒拆封声 |
| **场景钩** | 过境飞机配件、纯实验、仿制药报批 |
| **30 秒口述** | ③ 临时过境外国运输工具为自身需要在装置设备中使用有关专利；④ 专为科学研究和实验而使用；⑤ 为提供行政审批所需信息，制造、使用、进口专利药品或医疗器械，及专门为其制造、进口的 → 不视为侵权 |
| **案例锚** | **临时过境** · **科研实验** · **Bolar例外** |
| **商标对照** | 无直接对应；专利特有「政策例外舱」 |

### 房间 S5｜合法来源窗 · 第 77 条

| 项 | 内容 |
|----|------|
| **空间** | 与商标 64② 同构的窗户：进货单、上家信息；窗框写「不赔」 |
| **感官** | 发票油墨味 |
| **场景钩** | 使用者/销售者不知道侵权，能证明合法来源 |
| **30 秒口述** | 为生产经营目的使用、许诺销售或销售，不知道是未经许可制造并售出的专利侵权产品，能证明合法来源的，**不承担赔偿责任** |
| **要点** | 免赔不等于必然免停止侵害；行为类型限于使用/许诺销售/销售 |
| **案例锚** | **合法来源** · **免赔** · 不知情 |
| **商标对照** | ≈ 商标 64② / 4F·F5 来源窗 |

## 4. 侧窗与电梯

### 房间 W1｜禁止反悔 / 捐献窗（司法解释口径）

| 项 | 内容 |
|----|------|
| **空间** | 窗上贴着审查档案：为授权/维持效力限缩过的特征，不能等同扩张回来；说明书公开未写入权利要求的技术方案，视为捐献 |
| **感官** | 档案袋尘土；「限缩」印章 |
| **场景钩** | 等同主张被禁止反悔卡住；实施例没写进权要 |
| **30 秒口述** | 等同受**禁止反悔**限制：审批或无效程序中限缩放弃的，一般不能再用等同拿回来；说明书公开未要求保护的技术方案，可能适用**捐献规则**不得再主张等同纳入 |
| **案例锚** | **禁止反悔** · **捐献规则** · 限制等同 |
| **关联** | ↔ R2；细则进未来专利 B0 |
| **商标对照** | 商标少此机制；记作专利等同的「刹车」 |

### 房间 E1｜无效 / 评价报告电梯 · 第 66 条第二款（+ 2F 预告）

| 项 | 内容 |
|----|------|
| **空间** | 电梯按钮：「实用新型/外观评价报告」「启动无效宣告（2F）」 |
| **感官** | 电梯叮咚；报告纸张涩手 |
| **场景钩** | 实用/外观侵权案，法院要评价报告；或稳定性存疑要无效 |
| **30 秒口述** | 涉及实用新型或外观设计专利的，法院或管理部门可要求出具**专利权评价报告**作证据；当事人也可主动出具。专利稳定性不足时，常并行**无效宣告**（细节上 2F） |
| **案例锚** | **评价报告** · **稳定性** · 无效衔接 |
| **商标对照** | ≈ 商标异议/无效电梯，但专利民事中「评价报告」更日常 |

## 5. 楼梯口数字桩预告（通往 4F）

| 桩 | 预告 |
|----|------|
| **3** | 诉讼时效三年（74）；合法来源与赔偿抗辩常联动 |
| **5** | 71：故意+情节严重 → 惩罚性一至五倍；法定赔三万以上五百万以下 |
| **阶梯** | 71：损失或获利 → 许可费倍数 → 法定区间；加合理开支；举证妨碍 |

细拆留给专利 4F。

## 7. 通勤闭眼脚本（约 10～12 分钟）

1. **P1→P2**（2 分）：范围 + 实施行为  
2. **R1→R2**（3 分）：字面 → 等同（记得 W1 刹车）  
3. **R3 / R4**（2 分）：方法倒置或外观比对（按案选）  
4. **S1→S5**（3 分）：抗辩侧廊快过  
5. **E1 + 楼梯桩**（1 分）：稳定性与赔偿预告
