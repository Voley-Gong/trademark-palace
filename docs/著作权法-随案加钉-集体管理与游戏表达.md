## B15｜集体管理细则后室

**依据骨架：** 著作权法 §8；《著作权集体管理条例》§2 等；民事司法解释诉讼资格要点。

### 记忆四件事

1. 与使用者订立许可使用合同  
2. 向使用者收取使用费  
3. 向权利人转付使用费  
4. 以自己名义进行诉讼、仲裁等  

### 办案钩子

| 钩子 | 口径 |
|------|------|
| 诉讼资格 | 依法设立 + **书面授权** → 可以**自己名义**起诉 |
| 授权范围 | 先核权项与作品是否覆盖争议对象 |
| 一揽子许可 | **≠** 延伸性集体管理全覆盖；非会员作品仍可能另诉 |
| 组织地位 | 一般不享有原始著作权，靠授权行使 |

口诀：**四件事 + 书面授权；一揽子不是免死金牌。**

**通勤：** T2 → B15 → P2/F1

## B16｜游戏规则与表达边界柜

### 分拣表

| 通常不保护（思想/方法侧） | 通常可保护（表达侧） |
|---------------------------|----------------------|
| 玩法规则、胜负逻辑 | 美术角色、场景、UI 独创画面 |
| 数值、经济系统抽象设计 | 文案、对白、剧情具体表达 |
| 通用交互与操作方法 | 音乐、音效（符合作品要件时） |
| 「消除 / 卡牌 / 放置」等类型标签 | 有独创性的关卡具体编排 |
| | 计算机软件代码（另接软件条例） |

### 路径

1. O2 / S4：先问是不是受保护表达  
2. B16：分拣规则 vs 表达  
3. R1 / B10：仅对表达做接触 + 实质性相似  
4. 表达弱、市场仿冒强 → E2 换**竞争馆**（混淆 / 数据 / 虚假宣传）

口诀：**玩法像先过滤；著作权问表达相似。**

**通勤：** O2 → B16 → S4 → R1 →（弱则）E2

## 五问模板（两案）

**集体管理案：** ① 谁有书面授权？② 四件事做到哪步？③ 争议曲目是否在管理范围？④ 一揽子能否抗非会员？⑤ 诉请停还是赔？

**游戏案：** ① 抄的是规则还是表达？② 表达清单勾了哪些？③ 接触怎么证？④ 实质性相似过不过？⑤ 要不要改亮竞争馆？
