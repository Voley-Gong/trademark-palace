---

## 1. 总骨架（《电子商务法》）

| 条 | 动作 |
|----|------|
| §41 | 建立知识产权保护规则，与权利人合作 |
| §42 | 通知（初步证据）→ 及时必要措施 + 转送；未及时 → **损害扩大部分**连带；恶意通知可加倍赔偿 |
| §43 | 经营者反通知（不存在侵权初步证据）→ 转送权利人 |
| §44 | 转送反通知后，权利人未在 **十五日**内投诉/起诉 → 及时终止措施 |
| §45 | **知道或应当知道**侵权 → 应采取措施；未采取 → 与侵权人连带 |

必要措施：删除、屏蔽、断开链接、**终止交易和服务**等。  
行政：未依法采取措施可被责令改正并罚款（常见 §84 口径）。

## 3. 合流规则（办案）

1. 先定权利类型，再套电商法总骨架。  
2. 扩大部分连带 ≠ 全案连带；应知未措施才是全案连带常见表述。  
3. 版权：**避风港 ≠ 无通知即无责**。  
4. 刷单/仿冒装潢治理：别只会写电商法，记得反法 §21。  
5. 必要措施不止下架链接，含终止交易和服务。
