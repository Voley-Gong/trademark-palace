# 著作权法 · 3F 侵权认定层｜完整记忆宫殿房间表

> 依据：《中华人民共和国著作权法》（2020 年修正，2021 年 6 月 1 日起施行）及侵权比对实务口径  
> 定位：律所大楼「著作权馆」三楼——读案时「是不是作品、侵了哪项权、是否实质性相似、有无合理使用」  
> 对照：商标馆 3F / 专利馆 3F（见文末对照表）  
> 数据：`docs/data/copyright-palace.json`

---

## 0. 本层总图

```text
电梯口（争议定性：著作权？邻接权？还是反法/商标？）
    │
    ▼
前厅【作品门】P1 第3条 ──►【权利束】P2 第10条
    │
    ▼
主走廊（侵权判定）
  R1 接触+实质性相似厅 → R2 人身权侵害廊(52)
  → R3 财产权七剑(53①) → R4 邻接权门厅 → R5 技术措施/权利管理信息
    │         └─ N1–N8 邻接侧廊（出版/表演/录制/广播，见 `著作权法-邻接权层-记忆宫殿.md`）
    ├── 抗辩侧廊
    │     S1 合理使用(24) · S2 法定许可(25侧签)
    │     S3 合法来源/授权证明(59) · S4 思想表达二分(口径)
    │     S5 保护期届满(23预告1F)
    ├── 电梯【软件/信息网络条例】E1（第64条 → 2F/B0）
    └── 跨馆门【装潢/名称】E2 → 未来竞争馆 / 商标馆
    │
    ▼
楼梯 → 4F 救济（见 `著作权法-救济层-记忆宫殿.md`）
```

**本层只解决一件事：** 权利是否覆盖被控使用、是否构成侵权、有哪些法定限制与抗辩。  
权属（职务/委托/视听）→ 1F；程序与平台 / 软件条例 → 2F/B0（见 `著作权法-程序与条例层-记忆宫殿.md`）；赔多少 → 4F。

**跨馆口诀：**  
**商标问来源混淆，专利问特征落入，著作权问表达相似。**

## 1. 前厅

### 房间 P1｜作品门 · 第 3 条

| 项 | 内容 |
|----|------|
| **空间** | 铁门两侧：左挂「独创性」灯，右挂「一定形式表现」柜；门楣列文字/音乐/美术/视听/计算机软件等作品类型 |
| **感官** | 灯亮才开门（独创）；柜门冰凉（要能被感知的形式，不是纯思想） |
| **场景钩** | 被告称「这不是作品」「只是思想/流程/通用表达」 |
| **30 秒口述** | 作品＝文学、艺术和科学领域内具有**独创性**并能以**一定形式表现**的智力成果，包括文字、口述、音乐戏剧等、美术建筑、摄影、视听、工程设计图、地图示意图、计算机软件及法律规定的其他作品 |
| **要件** | ① 独创性 ② 可表现的形式 ③ 落入作品类型（或依法保护的其他智力成果形态） |
| **案例锚** | **独创性** · **一定形式表现** · **思想表达二分预告** |
| **关联** | → P2、R1；思想/通用元素抗辩 → S4 |
| **商标对照** | ≈ 商标 P1（56）：都是「先钉权利客体边界」 |
| **专利对照** | ≈ 专利 P1（64）：专利钉权要/视图，著作权钉「作品是否成立」 |

### 房间 P2｜权利束门 · 第 10 条

| 项 | 内容 |
|----|------|
| **空间** | 两排钥匙挂板：上排人身权四枚（发表/署名/修改/保护作品完整）；下排财产权十三枚（复制、发行、出租、展览、表演、放映、广播、**信息网络传播**、摄制、改编、翻译、汇编、其他） |
| **感官** | 信息网络传播权钥匙最烫（互联网案件高频）；人身权钥匙终身感 |
| **场景钩** | 起诉状列了哪些权？是复制还是信息网络传播？有没有人身权请求？ |
| **30 秒口述** | 著作权含人身权与财产权。办案先勾选被侵害的具体权项：网上提供作品使公众可在选定时间地点获得 → 常打**信息网络传播权**；制售盗版 → 复制权+发行权；短视频二创 → 可能涉及改编/摄制等。财产权可许可、转让并获酬 |
| **要件** | ① 有效权利（作者或继受/许可链条，细规 1F） ② 明确主张的权项 ③ 被控行为落入该权项控制范围 |
| **案例锚** | **权利束勾选** · **信息网络传播权** · **人身权并行** |
| **关联** | → R 廊；邻接权另见 R4 |
| **商标对照** | ≈ 商标 P2（48 使用）：一边问「是不是商标性使用」，一边问「侵了哪一项著作财产权/人身权」 |

## 2. 主走廊：侵权判定

### 房间 R1｜接触 + 实质性相似厅（司法口径）

| 项 | 内容 |
|----|------|
| **空间** | 双门闸机：左闸「接触或接触可能」，右闸「实质性相似」；中间比对台摊开两部作品 |
| **感官** | 闸机滴滴；比对台上荧光笔味（勾相同表达） |
| **场景钩** | 两部小说/两款游戏/两段代码像不像？被告说没看过原告作品 |
| **30 秒口述** | 著作权侵权比对常用公式：**接触（或接触可能）+ 实质性相似**。接触可用接触可能、合理机会推定；实质性相似比的是**表达**，不是思想、情感、创意或公有领域材料。软件、文字、美术、视听各有比对方法（细则进 B0） |
| **要件** | ① 原告作品受保护 ② 被告接触或有接触可能 ③ 表达层面实质性相似 ④ 相似部分属于保护范围 |
| **案例锚** | **接触** · **实质性相似** · **表达比对** |
| **商标对照** | ≈ 商标 R2 混淆：都是「不像一模一样也可能责」；著作权锚在表达相似，不在来源混淆 |
| **专利对照** | ≈ 专利 R1/R2：专利比技术特征，著作权比表达 |
| **数字桩** | 桩 **2**＝两部作品对照 |

### 房间 R2｜人身权侵害廊 · 第 52 条（一）～（五）等

| 项 | 内容 |
|----|------|
| **空间** | 走廊左侧五扇窄门：擅自发表 / 合作作品独吞署名 / 未参加创作强行署名 / 歪曲篡改 / **剽窃** |
| **感官** | 剽窃门有复印机过热味；署名门墨水溅手 |
| **场景钩** | 抄袭、洗稿、挂名、歪曲改编伤害作品完整 |
| **30 秒口述** | 第52条前段多涉人身权与特定使用：未经许可发表；合作作品当独创发表；未参加创作为名利署名；歪曲篡改；**剽窃**。还可连展览/摄制/改编翻译注释等未经许可使用（52⑥）及未付酬（52⑦）等。责任含停止侵害、消除影响、赔礼道歉、赔偿损失 |
| **案例锚** | **剽窃** · **署名权** · **保护作品完整** |
| **关联** | ↔ P2 人身权；赔礼道歉请求常挂这里 |
| **商标对照** | 商标少「赔礼道歉」人身面向；著作权人身权是特有廊 |

### 房间 R3｜财产权七剑厅 · 第 53 条第一项

| 项 | 内容 |
|----|------|
| **空间** | 七把剑挂墙上：**复制、发行、表演、放映、广播、汇编、信息网络传播**；剑柄烫印「未经许可」 |
| **感官** | 信息网络传播剑最常出鞘；复制剑金属冷 |
| **场景钩** | 网盘分享、聚合链接、盗版书、演唱会曲目、短视频搬运 |
| **30 秒口述** | 未经著作权人许可，**复制、发行、表演、放映、广播、汇编、通过信息网络向公众传播**其作品（法律另有规定除外）→ 承担第52条民事责任；同时损害公共利益的，还可行政查处、罚款，直至刑事。这是财产权侵权的主干剑阵 |
| **要件** | ① 未经许可（或无法定抗辩） ② 实施七类行为之一 ③ 对象为受保护作品 |
| **案例锚** | **复制发行** · **信息网络传播** · **行政+民事** |
| **数字桩** | 桩 **5**：违法经营额≥5万可并处一至五倍罚款（53 行政段）；与民事惩罚性区分场景 |
| **商标对照** | ≈ 商标 57 行为清单；著作权七剑更贴传播形态 |
| **关联** | → R5；抗辩 → S1；救济 → 4F·54 |

### 房间 R4｜邻接权侧厅 · 第 53 条（二）～（五）等

| 项 | 内容 |
|----|------|
| **空间** | 侧厅四柜：专有出版 / 表演者 / 录音录像制作者 / 广播电台电视台 |
| **感官** | 唱片塑料味；舞台追光热 |
| **场景钩** | 盗版碟、未经许可传表演、扒广播信号 |
| **30 秒口述** | 除作品著作权外，还要会认邻接权侵权：出版他人享有专有出版权的图书；未经表演者许可复制发行或网络传播其表演；未经录制者许可复制发行或网络传播录音录像制品；未经许可播放、复制或网络传播广播、电视等（详见 53②～⑤）。版式设计、现场直播表演等还可见 52⑨⑩ |
| **案例锚** | **专有出版权** · **表演者权** · **录音制作者权** |
| **关联** | 细则与合同 → 1F/2F；常与 R3 并行主张 |

### 房间 R5｜技术措施 / 权利管理信息厅 · 第 49、51、53 条（六）（七）

| 项 | 内容 |
|----|------|
| **空间** | 保险柜式房间：锁芯是「技术措施」；标签被撕处写「权利管理信息」 |
| **感官** | 破解工具金属磕碰；删ID标签的撕纸声 |
| **场景钩** | 提供破解补丁、去水印批量分发、删版权信息后上传 |
| **30 秒口述** | 权利人可采取技术措施。禁止故意避开或破坏技术措施，禁止以避开破坏为目的制造进口提供装置部件，禁止故意提供避开破坏的技术服务（法律另有规定可避开的除外）。亦禁止故意删改权利管理信息，或明知被删改仍向公众提供。对应 53⑥⑦，并可接行政/刑事 |
| **案例锚** | **避开技术措施** · **权利管理信息** · **提供破解工具** |
| **关联** | 可避开情形 → 50（B0）；平台责任 → E1/B0 |

## 3. 抗辩侧廊

### 房间 S1｜合理使用厅 · 第 24 条

| 项 | 内容 |
|----|------|
| **空间** | 十三格陈列柜（个人学习、适当引用、报道新闻、课堂教学……无障碍提供等）；柜顶横幅三句：**指明作者与作品名** · **不影响正常使用** · **不不合理损害合法权益** |
| **感官** | 引用格最拥挤；横幅字烫手（三要件总闸） |
| **场景钩** | 被告主张「我合理使用」「适当引用」「个人学习」 |
| **30 秒口述** | 符合第24条所列情形，可**不经许可、不付报酬**使用，但应指明作者姓名或名称、作品名称，且不得影响作品正常使用，也不得不合理地损害著作权人合法权益。前款也适用于对邻接权的限制。办案先对号入座情形，再过三句总闸 |
| **案例锚** | **合理使用** · **适当引用** · **三句总闸** |
| **商标对照** | ≈ 商标 59 正当使用窗；著作权合理使用清单更长、互联网争议更多 |
| **数字桩** | 可自建「24＝合理」谐音桩 |

### 房间 S2｜法定许可侧签 · 第 25 条（不单独精修）

| 项 | 内容 |
|----|------|
| **空间** | S1 旁小门「教科书汇编」 |
| **口述** | 为实施义务教育和国家教育规划编写出版教科书，可在法定范围内汇编已发表作品片段等，但应**付酬**、指明作者与作品名，并不得侵害其他权利。区别于合理使用：这里通常仍要付酬 |
| **案例锚** | **法定许可** · **付酬** · **教科书** |

### 房间 S3｜合法来源 / 授权证明窗 · 第 59 条

| 项 | 内容 |
|----|------|
| **空间** | 与专利/商标来源窗同构：出版者制作者要证明**合法授权**；发行者、软件/视听/音像复制品出租者要证明**合法来源** |
| **感官** | 授权合同纸涩；进货单油墨 |
| **场景钩** | 书店、网店卖盗版称不知情 |
| **30 秒口述** | 复制品的出版者、制作者不能证明出版、制作有合法授权的，复制品的发行者或视听作品、计算机软件、录音录像制品复制品的出租者不能证明发行、出租的复制品有合法来源的，应当承担法律责任 |
| **案例锚** | **合法授权** · **合法来源** · **发行者举证** |
| **商标/专利对照** | ≈ 商标 64② / 专利 77：都是流通环节举证；著作权 59 写得更「推定责任」 |

### 房间 S4｜思想表达二分窗（口径补丁，连 B0）

| 项 | 内容 |
|----|------|
| **空间** | 窗上贴半透明膜：膜外「思想、创意、方法、情感」滤掉；膜内才是受保护「表达」 |
| **感官** | 膜面反光，只看见具体句式、线条、代码结构 |
| **场景钩** | 「主题一样就算抄」「玩法一样就算侵」 |
| **30 秒口述** | 著作权保护表达，不保护思想、操作方法、技术方案本身（技术方案可能走专利）。通用场景、必要场景、公有领域材料通常剔除后再做实质性相似判断 |
| **案例锚** | **思想表达二分** · **必要场景** · **公有领域剔除** |
| **关联** | ↔ R1；软件/游戏细则 → B0 |

### 房间 S5｜保护期沙漏 · 第 23 条（预告 1F）

| 项 | 内容 |
|----|------|
| **空间** | 沙漏刻「作者终生+死后五十年」等；视听/法人作品另有起算 |
| **口述** | 财产权保护期届满 → 进入公有领域，复制传播不再构成对该财产权的侵权（人身权中署名等另有规则）。收案先看还在不在期 |
| **案例锚** | **保护期** · **公有领域** |
| **关联** | 细则上 1F |

## 5. 楼梯口数字桩预告（通往 4F）

| 桩 | 预告 |
|----|------|
| **0** | 公有领域/保护期届满 → 财产权请求落空 |

## 7. 通勤闭眼脚本（约 10～12 分钟）
