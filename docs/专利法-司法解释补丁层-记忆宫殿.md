# 专利法 · 地下室｜司法解释与裁判口径补丁层

> 定位：专利馆地下室——法条「怎么判」的补丁间  
> 用法：1F–4F 走通主干后，下井补裁判口径；网页楼层 **B0 / 补丁**  
> 主要挂钩：最高人民法院《关于审理侵犯专利权纠纷案件应用法律若干问题的解释》（2009）及其（二）（2016），以及惩罚性赔偿、行为保全等配套口径  
> 说明：本层为**学习汇编要点**，便于记忆与调取，不替代正式司法解释全文检索

## 1. 房间表

### 房间 B1｜保护范围与全面覆盖补丁库

| 项 | 内容 |
|----|------|
| **空间** | 地下室第一库房：墙上钉着「以权利要求为准」；每张权要复印件旁挂「全部技术特征」打孔器 |
| **感官** | 打孔器咔哒（特征一个都不能少）；权要纸边卷起（选哪一项权要） |
| **场景钩** | 多项权要主张哪一项？少一个特征还想字面侵权？功能性特征怎么解释？ |
| **30 秒口述** | 权利人应明确其主张的权利要求；有两项以上的，一般以主张的为准。发明实用新型落入保护范围，通常要求被诉技术方案包含与权要记载的**全部技术特征相同或等同**的特征（全面覆盖）。功能性特征一般按说明书实施例及其等同的具体实施方式确定。说明书及附图可以解释权要，但不能用来**修改**权要另立范围 |
| **要点清单** | ① 明确主张的权利要求 ② 全部技术特征原则 ③ 功能性特征落说明书 ④ 解释≠改写 |
| **案例锚** | **全面覆盖** · **权要选择** · **功能性特征** |
| **关联** | ← O6、P1、R1；→ B2（等同扩展） |
| **商标对照** | ≈ 商标核定范围钉死；专利钉的是**特征全集** |

### 房间 B2｜等同 · 禁止反悔 · 捐献补丁库（W1 加深）

| 项 | 内容 |
|----|------|
| **空间** | 三面镜廊：正面「三基本+联想到」；左镜「审查档案限缩＝禁止反悔」；右镜「说明书公开未写入权要＝捐献」 |
| **感官** | 镜面变形仍可认；档案袋尘土；捐献章「弃」字烫手 |
| **场景钩** | 字面差一特征能否等同；审查时改窄了还能等同拿回吗；实施例没写进权要 |
| **30 秒口述** | **等同**：与权要特征相比，基本相同手段、基本相同功能、基本相同效果，且本领域普通技术人员无需创造性劳动即可联想到 → 可视为等同落入。**禁止反悔**：授权或无效程序中通过修改或意见陈述**放弃**的技术方案，侵权诉讼中一般不得再用等同重新纳入；若限缩性修改/陈述被审查部门**明确否定**，通常不视为放弃。**捐献**：说明书公开但未写入权利要求的技术方案，权利人主张等同纳入的，一般不予支持 |
| **要点清单** | ① 等同三基本+联想到 ② 反悔以放弃为结果 ③ 明确否定则可能不构成放弃 ④ 捐献卡住「写了却没要」 |
| **案例锚** | **等同三基本** · **禁止反悔** · **捐献规则** |
| **关联** | ← R2、W1、33；抗辩方常提反悔证据 |
| **商标对照** | 商标少此「审查档案刹车」；专利等同必过本库 |

### 房间 B3｜现有技术 / 现有设计抗辩补丁库

| 项 | 内容 |
|----|------|
| **空间** | 盾牌库：一面刻「一项现有技术方案」；告示「相同或无实质性差异」；角落贴「别随便拼多篇」 |
| **感官** | 文献纸张味；盾面冰凉 |
| **场景钩** | 被告甩一篇对比文件；或想拼多篇现有技术；与无效程序怎么分工 |
| **30 秒口述** | 被诉落入保护范围的全部技术特征，与**一项**现有技术方案的相应特征相同或**无实质性差异**的，应认定属于现有技术，**不构成侵权**（外观同理看现有设计）。抗辩成立走侵权否定，不必先等无效；但评价报告负面、稳定性差时，常并行无效（见 B7）。拼凑多篇技术方案作现有技术抗辩，通常更严（学习口径：优先单篇+/-公知常识，细则随解释演进） |
| **要点清单** | ① 一项现有技术/设计 ② 相同或无实质性差异 ③ 抗辩成功＝不侵权 ④ 与无效策略可并行 |
| **案例锚** | **现有技术抗辩** · **无实质性差异** · **不构成侵权** |
| **关联** | ← S1、O3；↔ T8 |
| **商标对照** | 类似「公共领域」思路，但专利有更硬的特征比对公式 |

### 房间 B4｜外观比对与设计空间补丁库

| 项 | 内容 |
|----|------|
| **空间** | 环形比对台：授权视图 vs 被诉产品；灯牌「整体观察、综合判断」；侧柜写「设计空间」大小旋钮 |
| **感官** | 灯光均匀；旋钮拧紧＝设计空间小、更易近似 |
| **场景钩** | 外观是否相同近似；局部外观怎么比；设计空间大小影响宽容度 |
| **30 秒口述** | 外观侵权比对通常以一般消费者的知识水平和认知能力，对授权外观与被诉设计作**整体观察、综合判断**，看是否相同或近似。设计空间较大时，一般消费者不易察觉的较小区别可能仍不近似；设计空间较小时，较小区别也可能导致不相同近似（学习口径）。保护范围以图片照片为准，简要说明用于解释 |
| **要点清单** | ① 一般消费者视角 ② 整体观察综合判断 ③ 设计空间影响 ④ 视图为准 |
| **案例锚** | **整体观察** · **一般消费者** · **设计空间** |
| **关联** | ← R4、O4、O7 |
| **商标对照** | 都有「整体观察」，但参照主体与混淆/来源逻辑不同 |

### 房间 B5｜间接侵权与共同侵权补丁库

| 项 | 内容 |
|----|------|
| **空间** | 侧廊两扇门：教唆/帮助门；专用件仓库门（明知专用仍提供） |
| **感官** | 零件箱金属磕碰；「明知」灯闪 |
| **场景钩** | 只卖专用部件不卖整机；平台或配套厂是否构成帮助 |
| **30 秒口述** | 明知有关产品系专门用于实施专利的材料、设备、零部件、中间物等，未经许可为生产经营目的将该产品提供给他人实施专利的，常按帮助侵权等口径处理；教唆他人实施专利的，可能构成教唆侵权（学习口径，对接共同侵权规则）。直接实施仍回第11条行为清单 |
| **要点清单** | ① 明知 ② 专用或诱导实施 ③ 生产经营目的提供 ④ 与直接实施区分 |
| **案例锚** | **帮助侵权** · **教唆侵权** · **专用部件** |
| **关联** | ← P2；商标 R6 故意帮助可对照记忆 |
| **商标对照** | ≈ 商标 57⑥ 帮助；专利更强调「专用+明知」 |

### 房间 B6｜原告适格与许可维权补丁库

| 项 | 内容 |
|----|------|
| **空间** | 挂号窗口：专利权人 / 利害关系人；柜台抽屉按许可类型分格：独占、排他、普通 |
| **感官** | 挂号章咚；普通许可格几乎锁死（单独起诉难） |
| **场景钩** | 被许可人能不能单独起诉？共有人之一起诉？ |
| **30 秒口述** | 专利权人和利害关系人可起诉。利害关系人通常包括独占实施许可合同的被许可人；排他许可被许可人在专利权人不起诉时可起诉；普通许可被许可人一般仅在合同明确授权时可单独起诉（学习口径）。共有行使另回 1F·O10：无约定时单独实施/普通许可可以，其他处分要全体同意 |
| **要点清单** | ① 权利人+利害关系人 ② 独占/排他/普通分档 ③ 合同授权很关键 ④ 共有另看 14 |
| **案例锚** | **利害关系人** · **独占可诉** · **普通须授权** |
| **关联** | ← O13、O10；→ F1 |
| **商标对照** | 商标许可备案对抗；专利侧重**许可类型×起诉资格** |

### 房间 B7｜无效衔接 · 评价报告 · 中止补丁库

| 项 | 内容 |
|----|------|
| **空间** | 双轨站台：左轨「侵权诉讼」、右轨「无效宣告」；站牌「是否中止」；售票窗递「评价报告」 |
| **感官** | 列车交会轰鸣；中止铃刺耳；报告纸涩 |
| **场景钩** | 实用外观被诉，要不要中止等无效？评价报告负面怎么办？ |
| **30 秒口述** | 实用新型/外观侵权案，法院可要求或当事人主动出具**专利权评价报告**。被告常并行请求无效；是否**中止**侵权诉讼，视专利类型、稳定性、无效理由与证据等综合判断（发明通常更不易仅因无效而中止；实用外观更常见中止讨论——学习口径）。评价报告结论不利时，法院可能释明现有技术抗辩或无效路径。无效决定生效后的追溯规则回 2F·T10·47 |
| **要点清单** | ① 评价报告作证据 ② 无效可并行 ③ 中止看稳定性与类型 ④ 无效结果咬 T10 |
| **案例锚** | **评价报告** · **并行无效** · **中止裁量** |
| **关联** | ← T8–T11、E1；→ B3、S1 |
| **商标对照** | 商标确权行政诉讼与民事衔接；专利「评价报告+无效」更日常 |

### 房间 B8｜赔偿与保全口径补丁库

| 项 | 内容 |
|----|------|
| **空间** | 四格档案柜：故意与情节严重 / 账簿举证妨碍 / 合理开支 / 合法来源与停售 |
| **感官** | 账簿锁嘎吱；禁令红灯远处闪（连 F6） |
| **场景钩** | 惩罚性够不够？对方不交账本？合法来源还要不要承担合理开支？诉前禁令审查什么 |
| **30 秒口述** | **惩罚性**对接第71条：故意侵犯专利权且情节严重，可在基数上乘一至五倍；证据看重复侵权、侵权规模、隐匿证据、伪造毁损等（学习口径）。**举证妨碍**：权利人尽力举证且账簿主要在对方，可责令提供；拒不提供或虚假提供，可参考权利人主张定赔。**合理开支**计入赔偿；合法来源免赔后是否仍承担合理开支，实践与解释演进中有讨论，办案以最新有效口径为准。**诉前行为保全**对接第72条：侵权可能性、难以弥补损害、损益平衡、公共利益与担保 |
| **要点清单** | ① 故意+情节严重 ② 账簿令与妨碍 ③ 合理开支 ④ 保全四要素（学习） |
| **案例锚** | **惩罚性口径** · **举证妨碍** · **诉前保全** |
| **关联** | ← F3、F5、F6、S5 |
| **商标对照** | ≈ 商标 B4/B5/B7；专利惩罚性用词是**故意**不是恶意 |

## 2. 通勤闭眼脚本（约 10～12 分钟）

1. **B1→B2**（3 分）：全面覆盖 → 等同三闸（反悔/捐献）  
2. **B3→B4**（3 分）：现有技术抗辩公式；外观整体观察+设计空间  
3. **B5→B6**（2 分）：间接侵权；谁有资格起诉  
4. **B7→B8**（3 分）：评价报告/无效/中止；赔偿与保全口径  
5. **回到主楼点名**（30 秒）：W1、S1、E1、F3、T8  

---

## 3. 读案 30 秒假想

实用外观专利起诉经销商：先开 **B7** 评价报告与是否中止；经销商主张合法来源 → **B8+F5**；制造方主张现有设计 → **B3**；权利人把说明书实施例当等同往里塞 → **B2 捐献**；只告卖专用模具的配件商 → **B5**；普通被许可人单独起诉 → **B6** 先查合同授权。
