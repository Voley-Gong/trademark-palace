# 专利法 · 2F 申请审查与无效程序层｜完整记忆宫殿房间表

> 依据：《中华人民共和国专利法》（2020 年修正）  
> 定位：律所大楼「专利馆」二楼——申请日、优先权、公布实审、驳回复审、无效宣告的「时间与程序走廊」  
> 下接 1F 权属客体 · 上通 3F 侵权认定（稳定性常经本层）  
> 对照：商标馆 2F（`商标法-程序层-记忆宫殿.md`）

## 0. 本层总图

```text
电梯口（程序争议：还在申请？被驳回？要复审？要无效？要评价报告？）
    │
    ▼
申请段【申请日】T1 28 ──►【优先权】T2 29/30
    │         └─ 侧签：单一性 31 · 撤回 32 · 修改不超范围 33
    ▼
审查段（发明主线）
  【十八月公布】T3 34 →【三年实审请求】T4 35
  →【意见通知/视为撤回】T5 37 →【驳回】T6 38 →【复审电梯】T7 41
    │
    ▼
授权出口（回 1F·O1）39/40 公告生效
    │
    ▼
授权后纠错 / 诉讼配套
  【无效请求】T8 45 →【无效审理+诉讼】T9 46 →【自始空洞】T10 47
  【评价报告窗】T11 66②（实用/外观） · 【期满前终止】T12 44（年费/放弃）
```

**本层只解决一件事：** 走哪条程序、卡哪个期限、授权前后权利状态如何切换。  
实体适格回 1F；落入与救济上 3F/4F。

**与商标 2F 最大结构差：** 专利**没有**「初审公告后三个月异议窗」；授权后主要靠**无效宣告**挑战效力。专利无效请求主体是**任何人**，本法**无**商标那种「相对事由五年除斥」（驰名恶意例外那套）。

## 1. 申请段

### 房间 T1｜申请日柜台 · 第 28 条

| 项 | 内容 |
|----|------|
| **空间** | 柜台电子钟停在「收到申请文件之日」；邮寄格旁贴「寄出邮戳日」 |
| **感官** | 收件章咚；邮戳油墨味 |
| **场景钩** | 申请日怎么定？邮寄会不会晚一天？ |
| **30 秒口述** | 国务院专利行政部门**收到**专利申请文件之日为申请日；邮寄的，以**寄出的邮戳日**为申请日 |
| **案例锚** | **申请日** · **收到日** · **邮戳日** |
| **关联** | → T2（优先权日地位）；期限起算、现有技术对比日都钉这里 |
| **商标对照** | 商标也有申请日，但专利申请日还直接掐 **20/10/15 期限钟**（1F·O11） |

### 房间 T1b｜侧签三枚 · 第 31 / 32 / 33 条

| 条 | 一句话 | 感官标签 |
|----|--------|----------|
| **31** | 一件发明/实用原则上一项；同一总发明构思可合案。外观原则上一项；相似外观或成套可合案 | 合案订书机咔哒 |
| **32** | 授权前可随时撤回申请 | 撤回单撕纸声 |
| **33** | 发明实用修改**不得超出**原说明书和权利要求书记载范围；外观修改不得超出原图片照片表示范围 | 「超范围」红闸砸下 |

**办案钩：** 无效/侵权里谈「修改超范围」「禁止反悔」，常回这枚 33 红闸（再连 3F·W1）。

### 房间 T2｜优先权沙漏 · 第 29、30 条

| 项 | 内容 |
|----|------|
| **空间** | 两个沙漏：外国/本国优先权；发明实用标 **十二个月**，外观标 **六个月**；补材料铃分别响 |
| **感官** | 沙漏细沙；逾期铃「视为未要求优先权」 |
| **场景钩** | 海外先申请再进中国；本国先申再改进申请；声明与副本逾期 |
| **30 秒口述** | 自外国首次申请日起，发明/实用 **12 个月内**、外观 **6 个月内**就相同主题在中国申请，可享优先权（协议/条约/互惠）。本国同样有 12/6 个月优先权。要求发明实用优先权：申请时书面声明，并自首次申请日起 **16 个月内**交副本；外观：申请时声明，**3 个月内**交副本。未声明或逾期 → **视为未要求** |
| **数字桩** | **12 / 6** 优先权窗；**16 / 3** 交副本；与 1F 新颖性宽限「六个月」分场景 |
| **案例锚** | **涉外优先权** · **本国优先权** · **逾期失权** |
| **关联** | ↔ T1；影响现有技术判断的时间锚点 |
| **商标对照** | 商标优先权统一感更强（多为六个月）；专利按类型拆 12/6，且交副本期限不同 |

## 2. 审查段（发明主线；实用/外观走初审授权捷径）

> 实用新型、外观设计：初审无驳回理由 → 直接授权公告（1F·O1·40）。本段走廊以**发明**公布+实审为主线。

### 房间 T3｜十八月公布廊 · 第 34 条

| 项 | 内容 |
|----|------|
| **空间** | 长廊尽头电子屏「发明专利申请公布」，倒计时自申请日起满 **十八个月**；旁有「请求早日公布」按钮 |
| **感官** | 公布屏冷光；早日公布键发热 |
| **场景钩** | 什么时候公开、公开后别人实施怎么办 |
| **30 秒口述** | 发明申请经初步审查符合要求的，自申请日起满**十八个月**即行公布；申请人可请求**早日公布** |
| **数字桩** | **18 个月**公布（自建：十八开书＝公开） |
| **案例锚** | **十八月公布** · **早日公布** |
| **关联** | → 临时保护 1F·O2b·13；→ T4 实审 |
| **商标对照** | 商标「九个月初审公告」后开异议；专利公布后**不**开对世异议窗，而是进入实审/临时保护 |

### 房间 T4｜三年实审闸 · 第 35 条（+36 标签）

| 项 | 内容 |
|----|------|
| **空间** | 闸机标「申请日起三年内请求实质审查」；逾期灯大字「视为撤回」；部门也可自行实审 |
| **感官** | 闸机滴答三年；视为撤回蜂鸣刺耳 |
| **场景钩** | 忘了提实审、要不要主动实审、参考资料没交 |
| **30 秒口述** | 发明专利申请自申请日起**三年内**，可根据申请人随时请求进行实质审查；无正当理由逾期不请求 → 申请**视为撤回**。必要时部门可自行实审。请求实审时应提交申请日前有关参考资料；曾在外国申请的，部门可要求提交该国检索/审查资料，逾期无正当理由不交 → 视为撤回（36） |
| **数字桩** | 桩 **3**＝三年实审请求窗（与诉讼时效三年同桩不同场景） |
| **案例锚** | **三年实审** · **视为撤回** · **参考资料** |
| **关联** | → T5；实审延迟补偿 ↔ 1F·O11·42② |
| **商标对照** | 商标无「申请人不请求就撤」的同构闸；专利发明特有 |

### 房间 T5｜意见通知车间 · 第 37 条

| 项 | 内容 |
|----|------|
| **空间** | 双向传真台：审查意见通知书 vs 陈述意见/修改；墙上钟「指定期限」 |
| **感官** | 打印机吐意见书；逾期不答复的蜂鸣＝视为撤回 |
| **场景钩** | 收到 OA、要不要改权要、期限不够 |
| **30 秒口述** | 实审认为不符合本法规定 → 通知申请人在指定期限内**陈述意见**或**修改**；无正当理由逾期不答复 → 申请**视为撤回** |
| **案例锚** | **审查意见** · **指定期限** · **视为撤回** |
| **关联** | → T6；修改边界 → T1b·33 |

### 房间 T6｜驳回闸门 · 第 38 条

| 项 | 内容 |
|----|------|
| **空间** | 闸机落下：陈述或修改后仍不符合规定 → **予以驳回** |
| **感官** | 闸机金属撞击 |
| **场景钩** | 最终驳回通知到手 |
| **30 秒口述** | 申请人陈述意见或修改后，部门仍认为不符合本法规定的，应当**予以驳回** |
| **案例锚** | **驳回** · 仍不符合 |
| **关联** | → T7 复审；实体理由回 1F（22/25/26 等） |
| **商标对照** | ≈ 商标 T4 驳回闸 |

### 房间 T7｜复审电梯 · 第 41 条

| 项 | 内容 |
|----|------|
| **空间** | 驳回闸旁电梯：上车键「收到通知起**三个月**内请求复审」→ 部门复审决定 → 不服再「三个月内」起诉 |
| **感官** | 电梯按键滴滴；两段「三个月」钟并排走 |
| **场景钩** | 不服驳回怎么办；复审输了去哪 |
| **30 秒口述** | 不服驳回：自收到通知之日起**三个月内**向国务院专利行政部门请求**复审**；部门复审后作决定并通知。不服复审决定：自收到通知之日起**三个月内**向人民法院起诉 |
| **数字桩** | **3 个月**复审 + **3 个月**诉讼（双三段） |
| **案例锚** | **驳回复审** · **三月起诉** · 行政诉讼 |
| **关联** | ← T6；注意：专利复审窗口是**三个月**，商标驳回复审常记**十五日**——别串 |
| **商标对照** | 商标 T6：15 日复审；专利 T7：**3 个月**复审 |

### 授权出口牌（不另开精修房）

> 发明实审无驳回理由 → 39 授予公告生效；实用/外观初审无驳回理由 → 40 授予公告生效。  
> 闭眼点名回 **1F·O1**，然后本层转入「授权后挑战」。

## 3. 授权后：无效宣告（本层核心下半场）

### 房间 T8｜无效请求厅 · 第 45 条

| 项 | 内容 |
|----|------|
| **空间** | 授权公告铜匣旁敞开的大厅：门口牌子「**任何单位或者个人**」；墙上大字「授予不符合本法有关规定」 |
| **感官** | 门永不因「五年」自动上锁（与商标相对无效闸对比强烈） |
| **场景钩** | 被诉侵权要不要并行无效？谁有资格提？ |
| **30 秒口述** | 自公告授予专利权之日起，**任何单位或者个人**认为该专利权的授予不符合本法有关规定的，可以请求国务院专利行政部门宣告该专利权**无效** |
| **要件** | ① 授权公告后 ② 任何人可请求 ③ 理由＝授予不符合本法有关规定（三性、公开充分、修改超范围、客体排除等，细则进 B0） |
| **案例锚** | **任何人可无效** · **授权后随时（权利存续期内）** · **不符合本法** |
| **关联** | → T9；实体理由 ↔ 1F·O3–O6；诉讼中常与侵权并行 |
| **商标对照** | 商标分绝对无效（T9）与相对五年闸（T10）；专利本条**主体更宽、本法无五年相对闸**——口诀：**商标相对看五年，专利无效门常开** |

### 房间 T9｜无效审理厅 · 第 46 条

| 项 | 内容 |
|----|------|
| **空间** | 对席审台：请求人 vs 专利权人；墙上流程「及时审查 → 决定 → 登记公告」；侧门「三个月起诉 + 对方为第三人」 |
| **感官** | 决定书盖章；第三人传票纸涩 |
| **场景钩** | 无效决定多久出、输了怎么打行政诉讼、对方地位 |
| **30 秒口述** | 部门对无效请求应及时审查并作决定，通知双方；宣告无效的决定登记公告。不服无效或维持决定：自收到通知之日起**三个月内**起诉；法院应通知无效程序对方当事人作为**第三人**参加诉讼 |
| **案例锚** | **无效决定** · **三月起诉** · **对方为第三人** |
| **关联** | → T10；与侵权诉讼的审理顺序/中止口径挂 B0 |
| **商标对照** | ≈ 商标无效后的复审/诉讼通道，但专利直接部门决定 + 三月起诉 |

### 房间 T10｜自始空洞 · 第 47 条

| 项 | 内容 |
|----|------|
| **空间** | 专利证书化作空框，匾额「视为自始即不存在」；角落保险箱「已执行的一般不追溯」 |
| **感官** | 空框回声；保险箱锁咔哒；「恶意」「明显不公」两把钥匙烫手 |
| **场景钩** | 无效后，过去判的赔偿、已付许可费/转让费怎么办？ |
| **30 秒口述** | 宣告无效的专利权**视为自始即不存在**。对无效前已执行的侵权判决/调解书、已履行或强制执行的侵权行政处理、已履行的许可合同和转让合同，**原则上无追溯力**；但专利权人**恶意**给他人造成损失应赔偿。不返还赔偿金/使用费/转让费**明显违反公平**的，应全部或部分返还 |
| **案例锚** | **自始不存在** · **无追溯力** · **恶意赔偿** · **公平返还** |
| **关联** | ← T8/T9；影响 3F/4F 历史案件与和解评估 |
| **商标对照** | ≈ 商标 T11·47，几乎同构记忆 |

## 4. 诉讼配套与终止侧廊

### 房间 T11｜评价报告窗 · 第 66 条第二款

| 项 | 内容 |
|----|------|
| **空间** | 实用/外观维权前的玻璃窗：递出「专利权评价报告」；窗贴「法院或管理部门可要求；当事人也可主动出具」 |
| **感官** | 报告纸涩手；窗玻璃映出「未实审」字样 |
| **场景钩** | 实用外观起诉要不要先拿评价报告？稳定性够不够？ |
| **30 秒口述** | 涉及实用新型或外观设计专利的侵权纠纷，法院或管理专利工作的部门可要求出具专利权评价报告作证据；专利权人、利害关系人或被控侵权人也可主动出具 |
| **案例锚** | **评价报告** · **实用外观** · **稳定性证据** |
| **关联** | ↔ 3F·E1；常与 T8 无效策略并行 |
| **商标对照** | 商标无完全同构；记作专利实用/外观「初审授权」的配套体检窗 |

### 房间 T12｜期满前终止侧柜 · 第 44 条（与 1F·O11 互见）

| 项 | 内容 |
|----|------|
| **空间** | 年费投币口熄灭 / 书面放弃声明投入；登记公告灯亮「终止」 |
| **感官** | 投币口空空；放弃声明纸薄 |
| **30 秒口述** | 未按规定缴年费，或专利权人书面声明放弃 → 专利权在期限届满前**终止**，由部门登记公告 |
| **案例锚** | **年费终止** · **放弃终止** |
| **要点** | 终止 ≠ 无效：终止向后失效色彩浓；无效是自始不存在（T10）。收案先分清 |
| **关联** | ↔ 1F·O11；请求权基础排查 |

## 7. 与商标馆 2F 对照（跨馆电梯）

| 功能 | 商标馆 | 专利馆 |
|------|--------|--------|
| 申请日 | 申请文件 | T1 28 收到/邮戳 |
| 优先权 | T2 六个月为主 | T2 **12/6** 分类型 |
| 公开/公告 | T3 九个月初审公告 | T3 **十八月**发明公布 |
| 授权前公众挑战 | **T5 三月异议** | **无**（主要靠授权后无效） |
| 驳回复审期限 | 15 日 | **3 个月** |
| 注册/授权后挑战 | 无效 44/45（相对五年） | **T8 45 任何人**（本法无五年相对闸） |
| 自始不存在 | T11 47 | T10 47（同构） |
| 使用维持 | T12 撤三 | 无撤三；有年费终止 T12·44 |
| 初审权利体检 | — | **T11 评价报告**（实用/外观） |
