# 专利法 · 4F 救济层｜完整记忆宫殿房间表

> 依据：《中华人民共和国专利法》（2020 年修正）

> 上楼入口：3F 走廊尽头楼梯（见 `专利法-侵权认定层-记忆宫殿.md`）  
> 对照：商标馆 4F（`商标法-救济层-记忆宫殿.md`）

## 0. 本层总图

```text
3F 楼梯上来
    │
    ▼
接待台【路径分流】65：协商 / 起诉 / 管理专利工作的部门处理
    │
    ├── 左翼【行政停侵车间】65：责令停止 · 15日不服可行政诉讼 · 调解赔偿
    │         └─ 侧柜 69 查处/处理职权 · 70 重大影响/合并/跨区
    │
    ├── 中厅【赔偿殿】71
    │         损失或获利并排柜 → 许可费倍数 → 惩罚阁楼(故意×1～5)
    │         → 法定区间柜台(3万～500万) · 旁门举证抽屉
    │
    ├── 右翼【免赔/时效】
    │         F5 来源窗 77（↔3F·S5） · F4 时效钟房 74
    │
    ├── 急诊通道【诉前】72 财产保全+行为禁令 → 73 证据保全
    │
    └── 红灯厅【假冒专利】68（≠普通侵权！）责令改正公告·五倍罚款·刑事
```

**本层只解决一件事：** 走哪条救济路径、停不停、赔不赔、赔多少、过没过时效、来不来得及保全。  
侵权能不能成立 → 回 3F。  
**本层易混点：** 普通专利侵权行政（65）**没有**商标式「经营额五倍罚款」；罚款阶梯主要在 **假冒专利（68）**。

## 3. 中厅｜第 71 条（赔偿殿）——本层核心

> 把 71 条拆成同一大厅里的连续站位。注意与商标 63 的**第一顺位差异**：专利是「损失**或者**获利」并排优先，不是严格「先损后利」。

```text
站①损失柜 ∥ 站②获利柜（二者择一优先）
    → 站③许可费倍数
    → 站④惩罚阁楼(故意+情节严重 → 1～5倍)
    → 站⑤法定区间柜台(3万以上～500万以下)
         旁门：举证抽屉（账簿令）
```

### 房间 F3-①／②｜损失柜 ∥ 获利柜 · 第 71 条第一款前段

| 项 | 内容 |
|----|------|
| **空间** | 大厅入口两柜并排：左「权利人少赚了多少」，右「侵权人多赚了多少」；门楣写「**或者**」 |
| **感官** | 两台计算器同步滴答；「或者」霓虹闪一下 |
| **口述** | 赔偿额按权利人**实际损失**或者侵权人因侵权所获**利益**确定（立法表述为并列优先，实务择证据更强的一侧） |
| **桩** | 桩 **1**＝损失侧；桩 **2**＝获利侧；中间「或者」灯＝不要记成商标那种强制先损后利 |
| **商标对照** | 商标 63：损→利（顺序）；专利 71：损**或**利（并列入口） |

### 房间 F3-③｜许可费柜台 · 第 71 条第一款中段

| 项 | 内容 |
|----|------|
| **空间** | 第三柜台，放着专利许可合同复印件 |
| **感官** | 合同碳粉味 |
| **口述** | 损失或获利**难以确定** → **参照该专利许可使用费的倍数**合理确定 |
| **桩** | 桩 **3**＝第三站（许可费倍数） |

### 房间 F3-④｜惩罚性阁楼 · 第 71 条第一款惩罚段

| 项 | 内容 |
|----|------|
| **空间** | 窄楼梯通阁楼，门牌「**故意** × **情节严重**」（不是商标那句「恶意」） |
| **感官** | 上楼腿酸；阁楼热风扑面（故意） |
| **口述** | 对**故意**侵犯专利权且**情节严重**的，可在按上述方法确定数额的**一倍以上五倍以下**确定赔偿 |
| **要件** | ① 已有损失/获利/许可费等基数方法 ② 故意 ③ 情节严重 ④ 倍数 1～5 |
| **数字桩** | 桩 **5**＝五倍上限；与假冒 68「五倍罚款」同桩不同场景：一民事惩罚、一假冒行政罚款 |
| **案例锚** | **故意严重** · **惩罚性** · **一至五倍** |
| **商标对照** | 商标写「恶意」；专利写「故意」——面试/办案口头别串词 |

### 房间 F3-⑤｜法定区间柜台 · 第 71 条第二款

| 项 | 内容 |
|----|------|
| **空间** | 大厅尽端柜台，铜牌两行：「**三万元以上**」「**五百万元以下**」 |
| **感官** | 法槌轻敲；铜牌冰凉；「三万」小字要凑近才看见 |
| **口述** | 损失、获利、许可使用费**均难确定** → 法院依专利权类型、侵权行为性质和情节等，判给**三万元以上五百万元以下** |
| **数字桩** | 桩 **3**＝三万元起；桩 **5**＝五百万元封顶（与商标「仅有上限」不同，专利有**下限**） |
| **案例锚** | **法定赔偿** · **三万起** · **五百万封顶** |

### 房间 F3-旁｜举证抽屉 · 第 71 条第四款

| 项 | 内容 |
|----|------|
| **空间** | 获利柜旁铁抽屉，钥匙在侵权人裤袋 |
| **感官** | 抽屉锁嘎吱；空抽屉回声＝拒不提供 |
| **口述** | 权利人已尽力举证，账簿资料主要由侵权人掌握 → 法院可责令提供；不提供或提供虚假 → 可**参考权利人主张和证据**定赔 |
| **案例锚** | **举证妨碍** · **账簿令** · **采信权利人主张** |
| **关联** | ↔ F7 诉前证保；↔ 3F 方法倒置闸（都是举证战场，但规则不同） |

### 房间 F3-贴｜合理开支标签 · 第 71 条第三款

| 项 | 内容 |
|----|------|
| **空间** | 贴在每个柜台边的「附加收据」夹 |
| **口述** | 赔偿数额**还应当包括**权利人为制止侵权支付的**合理开支**（律师费、公证费等实务口径挂 B0） |
| **案例锚** | **合理开支** · **加在基数上** |

**71 条闭眼口诀（20 秒版）：**  
损或利 → 许可费倍数 →（故意严重则×1～5）→ 都没有就 **3万～500万**；加合理开支；账簿可责令。

## 4. 右翼｜免赔窗 + 时效钟

### 房间 F5｜来源窗 · 第 77 条（↔ 3F·S5）

| 项 | 内容 |
|----|------|
| **空间** | 与 3F·S5、商标 F5 窗②同构：进货单、发票、上家名片；窗框大字「不赔」 |
| **感官** | 发票油墨味；窗玻璃映出「制造」二字被划掉（行为范围） |
| **场景钩** | 使用/许诺销售/销售者称不知情且有合法来源 |
| **30 秒口述** | 为生产经营目的**使用、许诺销售或销售**，不知道是未经许可制造并售出的专利侵权产品，能证明**合法来源**的，**不承担赔偿责任** |
| **要点** | ① 免免责，不等于必然免停止侵害 ② 行为类型**不含制造** ③ 与商标 64② 同构记忆 |
| **案例锚** | **合法来源** · **免赔不免停** · **不含制造** |
| **关联** | ← 3F·S5（同一法条，3F 当抗辩点名，4F 当赔算入口） |
| **商标对照** | ≈ 商标 F5 窗②；专利**没有**商标 64①「三年未使用免赔」那扇休眠窗 |

### 房间 F4｜时效钟房 · 第 74 条

| 项 | 内容 |
|----|------|
| **空间** | 右翼钟楼：主钟刻「三年」；侧钟刻「临时保护使用费」 |
| **感官** | 钟摆沉重；知道侵权人身份的那一刻钟声骤响 |
| **场景钩** | 收案先问：过没过时效？临时保护期使用费怎么起算？ |
| **30 秒口述** | 侵犯专利权诉讼时效**三年**，自权利人/利害关系人**知道或应当知道侵权行为以及侵权人**之日起算。发明申请公布后至授权前使用未付适当使用费的，请求支付使用费的时效亦为三年，一般自知道或应当知道他人使用之日起算；若授权前已知，则自**授权之日**起算 |
| **要件** | ① 三年 ② 起算＝知行为+知侵权人 ③ 临时保护费第二款单独记 |
| **数字桩** | 桩 **3**＝三年时效（与 71「三万元起」同桩，分场景调用） |
| **案例锚** | **三年时效** · **知行为且知侵权人** · **临时保护费** |
| **商标对照** | 商标侧时效多靠民总则/民诉口径；专利 74 写进本法，单独开钟房 |

## 5. 急诊通道｜第 72、73 条（诉前）

### 房间 F6｜急症红门 · 第 72 条（财产保全 + 行为禁令）

| 项 | 内容 |
|----|------|
| **空间** | 大厅侧面急诊通道，红灯旋转；门后是「即将发货/即将开展」的卡车与展台 |
| **感官** | 警灯闪；引擎发动；手心出汗（难以弥补） |
| **场景钩** | 展会前夜、产线即将量产、证据与损害都等不起判决 |

| **要件** | ① 正在或即将实施（含妨碍实现权利） ② 难以弥补的损害 ③ 起诉前 ④ 依法（担保等挂民诉补丁） |

| **关联** | → 起诉后行为保全；地下室：民诉保全 |
| **商标对照** | ≈ 商标 F6（65）；专利措辞多了「妨碍实现权利」与作为/不作为 |

| 项 | 内容 |
|----|------|

| **要件** | ① 灭失/难取得风险 ② 与制止专利侵权相关 ③ 起诉前 |

## 6. 红灯厅｜第 68 条（假冒专利——与普通侵权分厅）

### 房间 F8｜假冒专利红厅 · 第 68 条

| 项 | 内容 |
|----|------|
| **空间** | 急诊旁独立厅：假专利号灯箱、虚假「专利产品」标贴被撕下示众；警徽门在深处 |
| **感官** | 灯箱忽明忽灭；公告栏油墨刺鼻；警徽反光 |
| **场景钩** | 产品标「已获专利」但无权/专利无效/张冠李戴；或伪造专利文件——**不一定等于技术方案落入某有效专利** |
| **30 秒口述** | **假冒专利**的，除民事责任外：负责专利执法的部门责令改正并予公告，没收违法所得，可处违法所得**五倍以下**罚款；无违法所得或违法所得在**五万元以下**，可处**二十五万元以下**罚款；构成犯罪的，依法追究刑事责任 |
| **要点** | ① 假冒专利 ≠ 专利侵权落入 ② 罚款数字像商标行政，但对象是「假冒」 ③ 刑民可并行 |
| **数字桩** | 桩 **5**：五倍罚款 + 五万门槛 +（联想）二十五万封顶 |
| **案例锚** | **假冒专利** · **五倍/二十五万** · **刑民并行** |
| **关联** | ↔ F2b·69（查处职权）；刑法具体罪名挂 B0 |
| **商标对照** | 约等于商标「假冒/制标」行政+刑事接口的合体；**不要**拿 68 去套普通落入侵权的行政罚款 |
