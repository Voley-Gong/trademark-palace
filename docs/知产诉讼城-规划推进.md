| 功能 | 商标 | 专利 | 著作权 |
|------|------|------|--------|
| 权利产生 | 核准注册 | 授权公告 | **自动产生**（2） |
| 权属默认 | 注册人 | 专利权人 | **作者**（11） |
| 职务 / 委托 | — | 6 / 8 | **18 / 19**（软件再看软 13） |
| 二楼主题 | 异议无效撤三 | 审查无效 | **通知删除 + 避风港** |
| 地下室 | 司法解释 | 司法解释 | **软件条例 + 网络条例** |
| 核心比对 | 混淆 | 字面/等同 | **接触+实质性相似** |
| 主抗辩 | 59 | 67/75/77 | **24 合理使用** |
| 赔偿梯子 | 63 | 71 | **54**（500 元～500 万） |

口诀补丁：

- **著作权问是谁的，先找作者，再查职务委托视听合同。**
- **通知删除冲流水线，明知应知出避风港；软件另开地下室。**
- **著作权赔偿像专利（或），行政罚款像商标（五倍）；法定下限记五百元。**
